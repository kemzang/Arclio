Birinchi ishga tushirishda yordam
Windows SmartScreen birinchi ishga tushirishda "Windows protected your PC" yoki "Unknown publisher" xabarini ko'rsatishi mumkin — Arroxy bepul va ochiq manbali dastur bo'lib, Windows qurilmalari pulli sertifikat bilan imzolanmagan. Bu Arroxy-Setup-*.exe va Arroxy-Portable-*.exe ikkisiga ham tegishli va Arroxy xavfli ekanligini anglatmaydi. More info tugmasini bosing, so'ng Run anyway tugmasini bosing. Arroxy'ni faqat rasmiy GitHub Releases sahifasidan yuklab oling — manba kodi ochiq, shuning uchun o'zingiz tekshirishingiz yoki qurishingiz mumkin.
Birinchi ishga tushirishda yordam
macOS birinchi ishga tushirishda noma'lum ishlab chiquvchi ogohlantirishini ko'rsatadi — Arroxy hali kod imzolanmagan. Ilova belgisini o'ng tugma bilan bosing → Ochish, so'ng muloqot oynasida Ochish tugmasini bosing. Faqat bir marta kerak.
Birinchi ishga tushirishda yordam
AppImage: faylni o'ng tugma bilan bosing → Xususiyatlar → Dastur sifatida bajarishga ruxsat, yoki terminalda chmod +x Arroxy-*.AppImage buyrug'ini ishga tushiring. Ishga tushmasa, libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) yoki fuse2 (Arch) o'rnating.

Flatpak: flatpak install --user Arroxy-*.flatpak, so'ng ilova menyusidan ishga tushiring yoki flatpak run io.github.antonio_orionus.Arroxy buyrug'ini bajaring.
