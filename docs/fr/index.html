Aide au premier lancement
Windows SmartScreen peut afficher "Windows protected your PC" ou "Unknown publisher" au premier lancement — Arroxy est gratuit et open source, et les builds Windows ne sont pas signés avec un certificat payant. Cela s'applique à la fois à Arroxy-Setup-*.exe et à Arroxy-Portable-*.exe et ne signifie pas qu'Arroxy est dangereux. Clique sur More info, puis sur Run anyway. Ne télécharge Arroxy que depuis la page officielle GitHub Releases — le code source est public, tu peux donc l'inspecter ou le compiler toi-même.
Aide au premier lancement
macOS affiche un avertissement développeur non identifié au premier lancement — Arroxy n'est pas encore signée. Clic droit sur l'icône → Ouvrir, puis cliquez sur Ouvrir dans la boîte de dialogue. Une seule fois.
Aide au premier lancement
AppImage : clic droit sur le fichier → Propriétés → Autoriser l'exécution comme programme, ou exécutez chmod +x Arroxy-*.AppImage dans un terminal. Si le lancement échoue, installez libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) ou fuse2 (Arch).

Flatpak : flatpak install --user Arroxy-*.flatpak, puis lancez depuis le menu des applications ou exécutez flatpak run io.github.antonio_orionus.Arroxy.

Ce qu'il fait

Tout ce que vous attendez, sans la moindre friction.

Collez une URL, choisissez une qualité, cliquez sur télécharger. C'est tout.

Jusqu'à 4K UHD

2160p, 1440p, 1080p, 720p — toutes les résolutions que YouTube et les autres sites supportés proposent, plus la conversion audio seule en MP3, M4A/AAC, Opus et WAV.

60 fps et HDR préservés

Les flux haute fréquence d'images et HDR passent exactement comme YouTube les encode — sans perte de qualité.

Playlists incluses

Colle l'URL d'une playlist, télécharge toute la liste ou coche seulement les vidéos que tu veux avant qu'Arroxy les mette en file.

Mises à jour automatiques

Arroxy garde yt-dlp à jour et embarque ffmpeg dans l’app — publie des correctifs chaque semaine à mesure que YouTube et les autres sites évoluent.

21 langues

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — détecte la vôtre automatiquement.

Multiplateforme

Builds natifs pour Windows, macOS et Linux — installeur, portable, DMG ou AppImage.

Sous-titres comme tu veux

Sous-titres manuels ou auto-générés en SRT, VTT ou ASS — à côté de la vidéo, intégrés dans un .mkv portable, ou rangés dans un dossier Subtitles/.

SponsorBlock intégré

Passe ou marque les segments sponsors, intros, outros, autopromos et plus — coupe-les avec FFmpeg ou ajoute simplement des chapitres. Ton choix, par catégorie.

Auto-remplissage presse-papiers

Copiez n'importe quel lien supporté n'importe où et Arroxy le détecte au moment où vous revenez — une invite de confirmation garde le contrôle. Activez ou désactivez dans Paramètres avancés.

Nettoyage auto des URLs

Les paramètres de tracking (si, pp, feature, utm_*, fbclid, gclid et plus) sont automatiquement retirés des liens collés, et les enveloppes youtube.com/redirect sont déballées — le champ URL affiche toujours le lien canonique.

Se réduit dans le tray

Fermer la fenêtre loge Arroxy dans la barre système. Les téléchargements continuent en arrière-plan — cliquez sur l'icône pour rouvrir la fenêtre, ou quittez depuis le menu de la barre.

Métadonnées & pochette intégrées

Titre, date de mise en ligne, artiste, description, pochette et marqueurs de chapitres écrits directement dans le fichier — sans fichiers annexes ni balisage manuel.

YouTube + 2000 sites

Au-delà de YouTube, Arroxy télécharge depuis plus de 2000 sites supportés par yt-dlp — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org et bien plus encore. L'audio seul et les sous-titres fonctionnent partout, pas seulement sur YouTube.

Confidentialité

Ce qu'Arroxy ne fait pas.

Traitement 100 % local. Pas de pub, pas d'upsell, pas de serveurs tiers — les fichiers vont directement de yt-dlp vers ton disque.

Pas de connexion requise

Le mode par défaut fonctionne sans compte Google ni connexion. Le support optionnel des cookies est disponible dans les Paramètres avancés pour les contenus à restriction d'âge ou réservés aux membres — désactivé par défaut.

Fichiers locaux uniquement

Les fichiers vont directement de yt-dlp vers le dossier que tu choisis. Rien ne passe par un serveur distant.

Télémétrie anonyme

Télémétrie anonyme via OpenPanel — un identifiant aléatoire par installation aide à compter les lancements, versions, OS et crashs ; pas d’URLs, titres, chemins de fichiers, infos de compte, fingerprinting ni données personnelles. Tes téléchargements, historique et fichiers ne quittent jamais ta machine.

Pas de pub, pas d'upsell

Licence MIT. Pas de niveau premium, pas de fonctions verrouillées, pas de bannières publicitaires, pas de dark patterns. Tout le pipeline tourne en local via yt-dlp + ffmpeg.

Installer

Choisissez votre canal.

Téléchargement direct ou tout gestionnaire de paquets majeur — tous mis à jour automatiquement à chaque release.

Winget Windows

Recommandé pour Windows 10/11. Se met à jour avec le système.

winget install AntonioOrionus.Arroxy

Scoop Windows

Installation portable via le bucket Scoop. Aucun droit admin requis.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Tap le cask, installation en une commande. Binaire universel (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Installation en sandbox. Télécharge le bundle .flatpak depuis les Releases, installe-le en une commande. Pas de configuration Flathub nécessaire.

flatpak install --user Arroxy-*.flatpak
