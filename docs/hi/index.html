Arroxy

मुफ़्त 4K YouTube (+ 2000 साइट) डाउनलोडर।
कोई विज्ञापन नहीं, कोई ब्लोट नहीं, कोई अपसेल नहीं।

Arroxy Windows, macOS और Linux के लिए एक मुफ़्त, MIT-लाइसेंस्ड डेस्कटॉप YouTube और 2000+ समर्थित साइटों का डाउनलोडर है। 4K HDR तक 60 fps में वीडियो डाउनलोड करता है। कोई विज्ञापन नहीं, कोई ब्लोट नहीं, कोई अपसेल नहीं — बस URL पेस्ट करें और शुरू करें।

कोई विज्ञापन नहीं कोई ट्रैकिंग नहीं Open source (MIT)

Arroxy Windows, macOS और Linux के लिए डेस्कटॉप ऐप है।

डाउनलोड करने के लिए अपने कंप्यूटर पर यह पेज खोलें।

GitHub पर हर लाइन ऑडिट करें।
पहले लॉन्च की मदद
पहले लॉन्च पर Windows SmartScreen "Windows protected your PC" या "Unknown publisher" दिखा सकता है — Arroxy मुफ़्त और ओपन-सोर्स है और Windows बिल्ड पेड सर्टिफ़िकेट से साइन नहीं हैं। यह Arroxy-Setup-*.exe और Arroxy-Portable-*.exe दोनों पर लागू होता है और इसका मतलब यह नहीं कि Arroxy असुरक्षित है। More info क्लिक करें, फिर Run anyway। Arroxy केवल आधिकारिक GitHub Releases पेज से डाउनलोड करें — सोर्स कोड सार्वजनिक है, इसलिए आप इसे ख़ुद जाँच सकते हैं या बना सकते हैं।
पहले लॉन्च की मदद
macOS पहले लॉन्च पर अज्ञात डेवलपर की चेतावनी दिखाता है — Arroxy अभी कोड-साइन्ड नहीं है। ऐप आइकन पर राइट-क्लिक → Open, फिर डायलॉग में Open क्लिक करें। बस एक बार।
पहले लॉन्च की मदद
AppImage: फ़ाइल पर राइट-क्लिक → Properties → Allow executing as program, या टर्मिनल में chmod +x Arroxy-*.AppImage चलाएँ। फिर भी न चले तो libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) या fuse2 (Arch) इंस्टॉल करें।

Flatpak: flatpak install --user Arroxy-*.flatpak चलाएँ, फिर ऐप मेन्यू से लॉन्च करें या flatpak run io.github.antonio_orionus.Arroxy चलाएँ।

यह क्या करता है

जो आप उम्मीद करते हैं वह सब, बिना किसी अड़चन के।

URL पेस्ट करें, गुणवत्ता चुनें, डाउनलोड पर क्लिक करें। बस इतना।

4K UHD तक

2160p, 1440p, 1080p, 720p — YouTube और अन्य समर्थित साइटें जो भी resolution देती हैं, साथ ही सिर्फ ऑडियो के लिए MP3, M4A/AAC, Opus और WAV conversion।

60 fps और HDR सुरक्षित

हाई फ्रेम-रेट और HDR स्ट्रीम बिल्कुल वैसी ही आती हैं जैसी YouTube एनकोड करता है — बिना गुणवत्ता के नुकसान।

Playlists भी

playlist URL पेस्ट करें, पूरी सूची डाउनलोड करें, या Arroxy के queue में डालने से पहले सिर्फ वे वीडियो चुनें जो आप चाहते हैं.

स्वचालित अपडेट

Arroxy yt-dlp को ताज़ा रखता है और ffmpeg ऐप के अंदर देता है — YouTube और अन्य साइटों के बदलते रहने के साथ हर हफ़्ते फ़िक्स जारी करता है।

क्रॉस-प्लेटफ़ॉर्म

Windows, macOS और Linux के लिए नेटिव बिल्ड — इंस्टॉलर, पोर्टेबल, DMG या AppImage।

सबटाइटल, आपके अंदाज़ में

SRT, VTT या ASS में मैनुअल या ऑटो-जेनरेटेड सबटाइटल — वीडियो के बग़ल में सेव करें, पोर्टेबल .mkv में एम्बेड करें, या Subtitles/ फ़ोल्डर में रखें।

SponsorBlock बिल्ट-इन

स्पॉन्सर सेगमेंट, इंट्रो, आउट्रो, सेल्फ-प्रोमो और अन्य को स्किप या मार्क करें — FFmpeg से काटें या बस चैप्टर जोड़ें। हर कैटेगरी के लिए आपकी पसंद।

क्लिपबोर्ड ऑटो-फ़िल

कहीं भी कोई भी समर्थित लिंक कॉपी करें और वापस स्विच करने पर Arroxy तुरंत पहचान लेता है — एक पुष्टि प्रॉम्प्ट नियंत्रण बनाए रखता है। Advanced settings में सक्षम या अक्षम करें।

URL को अपने आप साफ़ करें

ट्रैकिंग पैरामीटर (si, pp, feature, utm_*, fbclid, gclid और अन्य) पेस्ट किए गए लिंक से अपने आप हट जाते हैं, और youtube.com/redirect रैपर खुल जाते हैं — URL फ़ील्ड में हमेशा canonical लिंक दिखता है।

ट्रे में छुपाएं

विंडो बंद करने पर Arroxy सिस्टम ट्रे में चला जाता है। डाउनलोड बैकग्राउंड में जारी रहते हैं — ट्रे आइकन पर क्लिक करके विंडो वापस लाएं, या ट्रे मेनू से बाहर निकलें।

एम्बेडेड मेटाडेटा और कवर आर्ट

शीर्षक, अपलोड तिथि, कलाकार, विवरण, कवर आर्ट और चैप्टर मार्कर सीधे फ़ाइल में लिखे जाते हैं — कोई साइडकार फ़ाइल नहीं, कोई मैन्युअल टैगिंग नहीं।

YouTube + 2000 साइट

YouTube के अलावा, Arroxy 2000+ साइटों से भी डाउनलोड करता है जिन्हें yt-dlp सपोर्ट करता है — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org और भी बहुत कुछ। सिर्फ ऑडियो और सबटाइटल हर जगह काम करते हैं, न सिर्फ YouTube पर।

गोपनीयता

Arroxy जो नहीं करता है।

100% लोकल प्रोसेसिंग। कोई विज्ञापन नहीं, कोई अपसेल नहीं, कोई थर्ड-पार्टी सर्वर नहीं — फ़ाइलें yt-dlp से सीधे आपकी डिस्क पर जाती हैं।

कोई लॉगिन आवश्यक नहीं

डिफ़ॉल्ट मोड बिना किसी Google खाते या लॉगिन के काम करता है। आयु-प्रतिबंधित या केवल-सदस्य सामग्री के लिए Advanced settings में वैकल्पिक कुकी सपोर्ट उपलब्ध है — डिफ़ॉल्ट रूप से बंद।

केवल लोकल फ़ाइलें

फ़ाइलें yt-dlp से सीधे आपके चुने हुए फ़ोल्डर में जाती हैं। कुछ भी रिमोट सर्वर से नहीं गुज़रता।

गुमनाम टेलीमेट्री

OpenPanel के ज़रिए गुमनाम टेलीमेट्री — हर इंस्टॉल के लिए एक रैंडम ID लॉन्च, वर्शन, OS और क्रैश गिनने में मदद करती है; कोई URL, शीर्षक, फ़ाइल पाथ, अकाउंट जानकारी, फ़िंगरप्रिंटिंग या निजी डेटा नहीं। आपके डाउनलोड, इतिहास और फ़ाइलें कभी आपकी मशीन नहीं छोड़तीं।

कोई विज्ञापन नहीं, कोई अपसेल नहीं

MIT लाइसेंस। कोई प्रीमियम टियर नहीं, कोई फ़ीचर गेटिंग नहीं, कोई बैनर विज्ञापन नहीं, कोई डार्क पैटर्न नहीं। पूरी pipeline yt-dlp + ffmpeg के ज़रिए लोकल पर चलती है।

FAQ

अक्सर पूछे जाने वाले प्रश्न

इंस्टॉल

अपना चैनल चुनें।

सीधा डाउनलोड या कोई भी प्रमुख पैकेज मैनेजर — हर रिलीज़ के साथ स्वचालित रूप से अपडेट।

Winget Windows

Windows 10/11 के लिए अनुशंसित। सिस्टम के साथ ऑटो-अपडेट।

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop bucket के माध्यम से पोर्टेबल इंस्टॉल। एडमिन अधिकार आवश्यक नहीं।

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask टैप करें, एक कमांड से इंस्टॉल करें। यूनिवर्सल बाइनरी (Intel + Apple Silicon)।

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

सैंडबॉक्स्ड इंस्टॉल। Releases से .flatpak बंडल डाउनलोड करें, एक कमांड से इंस्टॉल। Flathub सेटअप ज़रूरी नहीं।

flatpak install --user Arroxy-*.flatpak
