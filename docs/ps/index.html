Arroxy

بې له کوکیز. بې له ننوتلو. بې له ماتو سیشنونو.

Arroxy د Windows، macOS، او Linux لپاره وړیا، MIT-جوازه ډیسکټاپ یوتیوب ډاونلوډر دی. تر 4K HDR پورې ویډیوګانې د 60 fps سره ډاونلوډ کوي — پرته له دې چې هیڅکله د Google حساب، براوزر کوکیز، یا ننوتلو غوښتنه وکړي.

Arroxy د Windows، macOS، او Linux لپاره ډیسکټاپ اپلیکیشن دی.

ډاونلوډ کولو لپاره دا پاڼه د خپل کمپیوټر کې پرانیزئ.

GitHub کې هره کرښه آډیټ کړئ.
د لومړي پیلولو مرسته
Windows SmartScreen ممکن د لومړي پیلولو پر مهال "Windows protected your PC" یا "Unknown publisher" وښیي — Arroxy وړیا او خلاصه سرچینه ده او د Windows جوړونې د پیسو ورکولو سند سره لاسلیک شوي نه دي. دا د دواړو Arroxy-Setup-*.exe او Arroxy-Portable-*.exe لپاره پلي کیږي او دا نه معنی لري چې Arroxy خطرناک ده. More info کلیک کړئ، بیا Run anyway. یوازې د رسمي GitHub Releases پاڼې نه Arroxy ډاونلوډ کړئ — سرچینه عامه ده، نو تاسو کولی شئ پخپله یې وڅیړئ یا جوړ کړئ.
د لومړي پیلولو مرسته
macOS د لومړي پیلولو پر مهال د ناپیژندلي پراختیاکار خبرداری ښیي — Arroxy لاهم کوډ لاسلیک شوی نه دی. د اپ آیکون باندې ښي کلیک → پرانیزئ، بیا د ډیالوګ کې پرانیزئ کلیک کړئ. یوازې یو ځل اړین دی.
د لومړي پیلولو مرسته
AppImage: د فایل باندې ښي کلیک → ملکیتونه → د پروګرام په توګه د چلولو اجازه ورکړئ، یا ټرمینل کې chmod +x Arroxy-*.AppImage چلوئ. که پیل بیا هم ناکام شو، libfuse2 (Ubuntu/Debian)، fuse-libs (Fedora)، یا fuse2 (Arch) نصب کړئ.

Flatpak: flatpak install --user Arroxy-*.flatpak، بیا د خپل اپ مینو نه پیل کړئ یا flatpak run io.github.antonio_orionus.Arroxy چلوئ.

یو URL پیسټ کړئ، کیفیت غوره کړئ، ډاونلوډ کلیک کړئ. بس همدومره.

تر 4K UHD پورې

2160p، 1440p، 1080p، 720p — هر resolution چې YouTube یې وړاندې کوي، او د یوازې آډیو لپاره MP3، M4A/AAC، Opus او WAV ته conversion هم.

60 fps او HDR ساتل کیږي

د لوړ فریم ریټ او HDR سټریمونه هغسې رسیږي لکه یوتیوب چې انکوډ کوي — بې له کوم د کیفیت له لاسه ورکولو.

Playlists هم

د playlist URL پیسټ کړئ، ټول لېست ډاونلوډ کړئ یا یوازې هغه ویډیوګانې وټاکئ چې تاسو یې غواړئ مخکې له دې چې Arroxy یې queue کړي.

چند پلیټفارمه

د Windows، macOS، او Linux لپاره اصلي جوړونې — نصبوونکی، portable، DMG، یا AppImage.

ژباړې، ستاسو د خوښې سره سم

لاسي یا اوتومات تولید شوي کیپشنونه د SRT، VTT، یا ASS کې — د ویډیو تر خوا خوندي کړئ، د یوه portable .mkv کې ځای پر ځای کړئ، یا د Subtitles/ فولډر کې کیږدئ.

SponsorBlock جوړ شامل

د سپانسر برخې، انترو، اوترو، ځان-پروموشنونه، او نور پریږدئ یا نښه کړئ — د FFmpeg سره یې پرې کړئ یا یوازې فصلونه اضافه کړئ. ستاسو انتخاب، د هرې کټګورۍ لپاره.

د کلیپ بورډ اوتومات ډکول

هر چیرې یوتیوب لینک کاپي کړئ او Arroxy هغه سمدلاسه کشفوي کله چې تاسو بیرته راشئ — د تایید پرامپت تاسو د کنټرول سره ساتي. د پرمختللو ترتیباتو کې فعال یا غیرفعال کړئ.

اوتومات د URLs پاکول

د تعقیب پیرامیټرونه (si، pp، feature، utm_*، fbclid، gclid، او نور) اوتومات له پیسټ شوو یوتیوب لینکونو نه لرې کیږي، او youtube.com/redirect لفافې پرانیستل کیږي — د URL ساحه تل اصلي لینک ښیي.

د ټرې کې پټیږي

د کړکۍ بندول Arroxy ستاسو د سیسټم ټرې ته لیږي. ډاونلوډونه د پس پرده چلیدل ادامه ورکوي — د کړکۍ بیرته راوړلو لپاره د ټرې آیکون کلیک کړئ، یا د ټرې مینو نه وتئ.

ځای پر ځای شوي میټاډیټا او آرټ

سرلیک، د اپلوډ نیټه، هنرمند، توضیح، د کور آرټ، او د فصل نښې مستقیم فایل کې لیکل کیږي — بې له سایدکار فایلونو، بې له لاسي ټاګ کولو.

YouTube + 2000 ویب پاڼې

د YouTube تر څنګ، Arroxy له 2000+ ویب پاڼو څخه چې yt-dlp یې ملاتړ کوي ډاونلوډ کوي — Vimeo، Twitch، Twitter/X، TikTok، SoundCloud، Bandcamp، Bilibili، BBC iPlayer، archive.org، او نور ډېر. یوازې غږ او سب ټایټلونه هرچېرته کار کوي، نه یوازې پر YouTube.

خصوصیت

هغه شی چې Arroxy نه کوي.

ډیری یوتیوب ډاونلوډران پایله کې ستاسو کوکیز غوښتنه کوي. Arroxy هیڅکله داسې نه کوي.

بې له ننوتلو

بې له د Google حساب. بې له د پای ته رسیدلو سیشنونو. د حساب د نښه کیدو صفر خطر.

بې له کوکیز

Arroxy هغه ټوکنونه غوښتنه کوي چې هر براوزر کوي. هیڅ صادر شوي، هیڅ ذخیره شوي.

ناپیژندلې ټیلیمیټري

د OpenPanel له لارې ناپیژندلې ټیلیمیټري — د هر نصب تصادفي ID د پیلونو، نسخو، OS او کریشونو شمېرلو کې مرسته کوي؛ URLs، سرلیکونه، د فایل لارې، د اکاونټ معلومات، fingerprinting یا شخصي معلومات نشته. ستاسو ډاونلوډونه، تاریخچه او فایلونه هیڅکله ستاسو ماشین نه پرېږدي.

بې له دریم ډلې سرورونو

ټوله پایپلاین د yt-dlp + ffmpeg له لارې سیمه ایز چلیږي. فایلونه هیڅکله ریموټ سرور ته نه رسیږي.

FAQ

نصب کول

خپل چینل غوره کړئ.

مستقیم ډاونلوډ یا هر لوی پیکیج مینیجر — ټول د هرې خپرونې سره اوتومات تازه کیږي.

Winget Windows

د Windows 10/11 لپاره وړاندیز شوی. د سیسټم سره اوتومات تازه کیږي.

winget install AntonioOrionus.Arroxy

Scoop Windows

د Scoop bucket له لارې portable نصب. د اډمن حقونو اړتیا نشته.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

cask ټپ کړئ، د یو امر سره نصب کړئ. Universal binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Sandboxed نصب. د Releases نه .flatpak بنډل ډاونلوډ کړئ، د یو امر سره نصب کړئ. د Flathub ترتیب اړتیا نشته.

flatpak install --user Arroxy-*.flatpak
