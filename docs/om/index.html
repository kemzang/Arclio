Arroxy

Buufata YouTube 4K bilisaa.
Kuukii hin jiru. Seensa hin jiru. Seesiini cabuu hin jiru.

Arroxy buufata YouTube desktop bilisaa, hayyama MIT, Windows, macOS, fi Linux dhaaf dha. Viidiyoo hamma 4K HDR sa'aatii 60 fps buufa — akkaawuntii Google, kuukii browser, ykn seensa kamiiyyuu gaafachuu gonkumaa malee.

Akkaawuntii Google hin barbaadu Hordoffii hin jiru Open source (MIT)

Arroxy app desktop Windows, macOS, fi Linux dhaaf ta'edha.

Buufachuuf fuula kana kompyuutara keetii irratti bani.

Sarara koodii hunda GitHub irratti qoradhu.
Gargaarsa jalqaba banuu
Windows SmartScreen jalqaba banuu irratti "Windows protected your PC" ykn "Unknown publisher" agarsiisuu danda'a — Arroxy bilisaa fi madda banaa dha, ijaarsonni Windows mallattoo kafaltii hin qabne waliin hin mallatteffamne. Kun Arroxy-Setup-*.exe fi Arroxy-Portable-*.exe lameeniifuu hojjeta, Arroxy nageenya hin qabu jechuuf miti. More info cuqaasi, kana booda Run anyway cuqaasi. Arroxy fuula GitHub Releases mirkaneessaatii qofa buufi — koodii ummataadhaan banaa dha, kanaafuu of irraadhaa sakatta'uu ykn of irraadhaa ijaaru dandeessa.
Gargaarsa jalqaba banuu
macOS jalqaba banuu irratti guddifataa hin beekamne jechuun dhorka — Arroxy ammalee mallattoo koodii hin qabu. Akaawuntii app irratti cuqaasi mirgaan → Bani, kana booda galmee keessaa Bani cuqaasi. Yeroo tokkoo qofa barbaachisa.
Gargaarsa jalqaba banuu
AppImage: faylii irratti mirgaan cuqaasi → Amaloota → Akka sagantaatti hojjechuuf hayyami, ykn terminal keessaa chmod +x Arroxy-*.AppImage hojjeti. Yoo ammallee hin jalqabne, libfuse2 (Ubuntu/Debian), fuse-libs (Fedora), ykn fuse2 (Arch) diriirsi.

Flatpak: flatpak install --user Arroxy-*.flatpak, kana booda menu app keetii irraa bani ykn flatpak run io.github.antonio_orionus.Arroxy hojjeti.

Maal hojjeta

Waan eegdu hunda, rakkoo tokko malee.

URL maxxansi, qulqullina filadhu, buufi cuqaasi. Kun dha.

Hamma 4K UHD

2160p, 1440p, 1080p, 720p — resolution YouTube dhiyeessu hunda, akkasumas audio qofa gara MP3, M4A/AAC, Opus fi WAV jijjiiruu.

60 fps fi HDR eegamaniiru

Raadiyoon sarara fps olaanaa fi HDR YouTube encode godhe sana sana dhufu — qulqullina dhabuu tokko malee.

Playlist illee

URL playlist maxxansi, tarree guutuu buufadhu yookaan Arroxy queue keessa galchuu isaa dura viidiyoowwan ati barbaaddu qofa mallatteessi.

Haaromsa ofii

Arroxy yt-dlp haaromsee eega, ffmpeg immoo app keessatti waliin dhufa — jijjiirama YouTube hundaaf hojjeta.

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — kan kee ofii isaa argata.

Platform hedduu

Ijaarsi asaasaa Windows, macOS, fi Linux dhaaf — diriirfataa, portable, DMG, ykn AppImage.

Caaltuu, akka fedhetti

Caaltuu harka ykn ofumaan uumame SRT, VTT, ykn ASS keessatti — viidiyoo cinaa ol-kaa'ama, .mkv portable keessa ramadama, ykn gita Subtitles/ keessa kaa'ama.

SponsorBlock keessatti jira

Kutaa beeksisaa, seensa, baxxannoo, of-beeksisuu fi kkf irra ture ykn mallattoo isaan irratti kaa'i — FFmpeg waliin muri ykn boqonnaa dabalii. Filannoon kee, kutaa kutaadhaan.

Clipboard ofumaan guuta

Liinki YouTube eessallee copy godhi, Arroxy yeroo deebitu ni argata — mirkaneessa beeksisa to'annaa siif kenna. Qindaa'ina Olaanaa keessatti eegi ykn dhaabi.

URL ofumaan qulqulleessa

Parameetaroonni hordoffii (si, pp, feature, utm_*, fbclid, gclid, fi kanneen biroo) liinkii YouTube maxxanfame irraa ofumaan haaqamu, fi maramtoota youtube.com/redirect hiikamu — dirreen URL yeroo hunda liinkii bu'uuraa agarsiisa.

Tireey keessatti dhossata

Foddaa cufuu Arroxy tireey sirna keessatti ukaa'a. Buufachuu duubaan hojjechaa tura — akaawuntii tireey cuqaasii foddaa deebi'i, ykn menu tireey irraa ba'i.

Metadata fi barreeffama keessaa jiru

Mata-duree, guyyaa olkaa'uu, artisti, ibsa, barreeffama cufaa, fi mallattoolee boqonnaa kallattumaan faayilii keessa barreeffama — faayilii dhibbaa tokko malee, mallattoo of danda'aa malee.

YouTube + sayitii 2000

YouTube cinaatti, Arroxy sayitii 2000+ kan yt-dlp deggeru irraa fida — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org fi kanneen biroo hedduu. Sagalee qofaa fi sabtaayitiloonni iddoo hundatti hojjetu, YouTube qofa irratti miti.

Dhuunfaa

Arroxy hin hojjennu.

Buufatoota YouTube hedduun yeroo tokkotti kuukii kee gaafatu. Arroxy gaafachuu hin danda'u.

Seensa hin barbaadu

Akkaawuntii Google hin barbaadu. Seesiini dhumuu hin jiru. Akkaawuntiin kee mallatteffamuuf carraan duwwaa dha.

Kuukii hin jiru

Arroxy tokeenota browser kamiyyuu gaafatu sana gaafata. Wanti ol-baafame hin jiru, wanti kuufame hin jiru.

Teelemeetrii maqaa-dhabduu

Teelemeetrii maqaa-dhabduu OpenPanel waliin — ID tasaa kan install tokkoon tokkoo eegaluu, version, OS fi kufaatii lakkaaʼuuf gargaara; URL, mata-duree, karaa faayilaa, odeeffannoo account, fingerprinting yookaan data dhuunfaa hin jiru. Buufatoonni, seenaa fi faayiloonni kee meeshaa kee irraa hin bahan.

Sirnaalee sadaffaa hin jiran

Tooftaan guutuun naannoo yt-dlp + ffmpeg waliin hojjeta. Faayiloonni sarara fagoo tokkollee hin tuqan.

FAQ

Gaaffilee yeroo baay'ee gaafataman

Diriirsi

Karaa kee filadhu.

Kallattiin buufi ykn maniijara paakeejii guddaa kamiyyuu — hundi baasii hunda waliin ofumaan haaromfamu.

Winget Windows

Windows 10/11 dhaaf gorfama. Sirna waliin ofumaan haaromfama.

winget install AntonioOrionus.Arroxy

Scoop Windows

Diriirsa portable Scoop bucket waliin. Mirga bulchaa hin barbaadu.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask naqii, ajaja tokkoon diriirsi. Baayinariin universal (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Diriirsa sandbox. Xiinxala .flatpak Releases irraa buufi, ajaja tokkoon diriirsi. Qindaa'ina Flathub hin barbaadu.

flatpak install --user Arroxy-*.flatpak

Kallattiin buufi Hunda

Diriirfataa NSIS, .exe portable, .dmg, .AppImage, ykn .flatpak — kallattumaan GitHub Releases irraa.
