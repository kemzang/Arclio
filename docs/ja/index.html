Arroxy

無料 4K YouTube（+ 2000 サイト）ダウンローダー。
広告なし、余計なものなし、アップセルなし。

Arroxy は Windows、macOS、Linux 向けの無料・MIT ライセンスのデスクトップダウンローダーです。YouTube と 2000 以上の対応サイトから最大 4K HDR・60 fps で動画をダウンロード。広告なし、余計なものなし、アップセルなし — URL を貼り付けるだけで始められます。

広告なし トラッキングなし オープンソース (MIT)

Arroxy は Windows、macOS、Linux 向けのデスクトップアプリです。

ダウンロードするには、お使いのパソコンでこのページを開いてください。

GitHub ですべてのコードを監査できます。
初回起動のヘルプ
初回起動時、Windows SmartScreen が «Windows protected your PC» または «Unknown publisher» を表示することがあります — Arroxy は無料のオープンソースソフトウェアであり、Windows ビルドには有料の証明書によるコード署名がありません。これは Arroxy-Setup-*.exe と Arroxy-Portable-*.exe の両方に該当し、Arroxy が危険であることを意味しません。More info をクリックし、続いて Run anyway をクリックしてください。Arroxy は必ず公式の GitHub Releases ページからダウンロードしてください — ソースコードは公開されているため、ご自身で確認またはビルドすることも可能です。
初回起動のヘルプ
macOS は初回起動時に 開発元未確認 の警告を表示します — Arroxy はまだ署名されていません。アプリアイコンを右クリック → 開く、ダイアログで 開く をクリック。最初の一度だけです。
初回起動のヘルプ
AppImage: ファイルを右クリック → プロパティ → プログラムとして実行を許可、またはターミナルで chmod +x Arroxy-*.AppImage を実行。それでも起動しない場合は libfuse2 (Ubuntu/Debian)、fuse-libs (Fedora)、または fuse2 (Arch) をインストール。

Flatpak: flatpak install --user Arroxy-*.flatpak を実行し、アプリメニューから起動するか flatpak run io.github.antonio_orionus.Arroxy を実行。

できること

期待されることはすべて、面倒は一切なく。

URL を貼り付けて、画質を選んで、ダウンロードをクリック。それだけ。

最大 4K UHD

2160p、1440p、1080p、720p — YouTube と他の対応サイトが提供するあらゆる解像度に対応し、音声のみのダウンロードは MP3、M4A/AAC、Opus、WAV へ変換できます。

60 fps と HDR を維持

高フレームレートと HDR ストリームは YouTube がエンコードしたまま — 画質劣化なし。

プレイリストにも対応

プレイリスト URL を貼り付けて、一覧全体をダウンロードすることも、Arroxy がキューに入れる前に欲しい動画だけにチェックを入れることもできます。

自動アップデート

Arroxy は yt-dlp を最新に保ち、ffmpeg はアプリに同梱 — YouTube や他のサイトの変化に合わせて毎週修正を提供します。

21 言語対応

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — あなたの言語を自動検出。

マルチプラットフォーム

Windows、macOS、Linux のネイティブビルド — インストーラー、ポータブル、DMG、AppImage。

字幕は思いのままに

手動または自動生成の字幕を SRT、VTT、ASS で取得 — 動画と並べて保存、ポータブルな .mkv に埋め込み、または Subtitles/ フォルダに整理。

SponsorBlock 内蔵

スポンサー、イントロ、アウトロ、自己宣伝などをスキップまたはマーク — FFmpeg でカットするか章として追加するか。カテゴリごとにあなたが決める。

クリップボード自動入力

対応リンクをどこかでコピーするだけで、アプリに切り替えた瞬間 Arroxy が検出します — 確認プロンプトで常に制御を維持。詳細設定で有効・無効を切り替え。

URL 自動クリーンアップ

貼り付けたリンクからトラッキングパラメータ（si、pp、feature、utm_*、fbclid、gclid など）が自動で除去され、youtube.com/redirect ラッパーも展開 — URL フィールドには常に正規リンクが表示されます。

トレイに格納

ウィンドウを閉じると Arroxy はシステムトレイに格納されます。ダウンロードはバックグラウンドで継続 — トレイアイコンをクリックするとウィンドウが戻り、トレイメニューから終了もできます。

メタデータ・カバーアート埋め込み

タイトル、アップロード日、アーティスト、説明、カバーアート、チャプターマーカーをファイルに直接書き込み — サイドカーファイルも手動タグ付けも不要。

YouTube + 2000 サイト

YouTube 以外にも、Arroxy は yt-dlp が対応する 2000 以上のサイトからダウンロードできます — Vimeo、Twitch、Twitter/X、TikTok、SoundCloud、Bandcamp、Bilibili、BBC iPlayer、archive.org など多数。音声のみと字幕は YouTube に限らず、すべての対応サイトで機能します。

実際の動作

明快さのために設計、雑然さは排除。

プライバシー

Arroxy がしないこと。

100% ローカル処理。広告なし、アップセルなし、サードパーティサーバーなし — ファイルは yt-dlp からあなたのディスクに直接保存されます。

ログイン不要

デフォルトモードは Google アカウントやログインなしで動作します。年齢制限付きやメンバー限定コンテンツ向けのオプション Cookie サポートが詳細設定で利用可能 — デフォルトはオフ。

ローカルファイルのみ

ファイルは yt-dlp からあなたが選んだフォルダへ直接保存されます。リモートサーバーを経由しません。

匿名テレメトリー

OpenPanel による匿名テレメトリー — インストールごとのランダム ID は起動数、バージョン、OS、クラッシュの集計にだけ使われます。URL、タイトル、ファイルパス、アカウント情報、フィンガープリンティング、個人データはありません。ダウンロード、履歴、ファイルはすべてあなたのデバイス内に留まります。

広告なし、アップセルなし

MIT ライセンス。プレミアム層なし、機能ゲートなし、バナー広告なし、ダークパターンなし。全パイプラインが yt-dlp + ffmpeg でローカル実行。

FAQ

よくある質問

どの画質でダウンロードできますか？
YouTube と他の対応サイトが提供するものなら何でも対応しています。4K UHD（2160p）、1440p QHD、1080p Full HD、720p、480p、360p、そして音声のみ。高フレームレートのストリーム（60 fps、120 fps）や HDR コンテンツもそのまま保持されます。Arroxy は利用可能な形式をすべて表示し、音声のみのダウンロードでは MP3、M4A/AAC、Opus、WAV への変換も選べます。
本当に無料ですか？
はい。MIT ライセンス。プレミアム版や機能ロックはありません。
Arroxy は何カ国語に対応していますか？
21 言語に標準対応：English、Español（スペイン語）、Deutsch（ドイツ語）、Français（フランス語）、日本語、中文（中国語）、Русский（ロシア語）、Українська（ウクライナ語）、हिन्दी（ヒンディー語）、Afaan Oromoo、Kiswahili、O'zbekcha（ウズベク語）、Tiếng Việt（ベトナム語）、አማርኛ（アムハラ語）、العربية（アラビア語）、اردو（ウルドゥー語）、پښتو（パシュトー語）、বাংলা（ベンガル語）、မြန်မာဘာသာ（ビルマ語）、Ελληνικά（ギリシャ語）、Српски（セルビア語）。Arroxy は初回起動時に OS の言語を自動検出し、ツールバーの言語選択でいつでも切り替え可能です。翻訳は src/shared/i18n/locales/ 内の素の TypeScript オブジェクトとして管理されているので、GitHub で PR を開いて貢献できます。
何かインストールが必要ですか？
いいえ。yt-dlp は初回起動時に自動ダウンロードされてマシンにキャッシュされます。ffmpeg と ffprobe はアプリに同梱されています。それ以降は追加のセットアップ不要です。
YouTube が変更したら使えなくなりますか？
はい — Arroxy には 2 段階の耐性があります。第一に、yt-dlp はオープンソースで最も活発に保守されているツールの 1 つで、YouTube を含む対応サイトの変更から数時間以内に更新されます。第二に、Arroxy は Cookie や Google アカウントに一切依存しないため、失効するセッションも、ローテートする資格情報もありません。この組み合わせにより、ブラウザの Cookie に依存するツールよりも遥かに安定しています。
プレイリストはダウンロードできますか？
はい。プレイリスト URL を貼り付け、すべての動画または必要な動画だけを選ぶと、Arroxy がひとつのバッチとしてキューに入れます。チャンネル全体の一括ダウンロードはまだ未対応です。
YouTube アカウントや Cookie が必要ですか？
デフォルトでは不要です — Arroxy はアカウント、ログイン、Cookie のエクスポートなしで動作します。年齢制限付きやメンバー限定コンテンツなど、認証が必要なコンテンツ向けに、詳細設定でオプションの Cookie サポート（ファイルまたはブラウザインポート）が利用可能です。デフォルトはオフです。有効にする場合、yt-dlp の公式ドキュメントは Cookie ベースの自動化が Google アカウントにフラグを立てる可能性があると警告しているため、使い捨てアカウントを使うのが安全です。
macOS で「アプリが壊れている」「開けません」と表示される — どうすれば？
これは macOS Gatekeeper が未署名アプリをブロックしているもので、実際に壊れているわけではありません。README に macOS 初回起動のステップごとの手順があります。
これは合法ですか？
個人使用のための動画ダウンロードは、ほとんどの法域で一般的に容認されています。YouTube の利用規約および所在地の法律の遵守はあなたの責任です。
インストール

チャンネルを選んでください。

直接ダウンロード、または主要パッケージマネージャー経由 — すべてリリースごとに自動更新。

Winget Windows

Windows 10/11 推奨。システムと一緒に自動更新。

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop バケット経由のポータブルインストール。管理者権限不要。

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask を tap してワンコマンドでインストール。ユニバーサルバイナリ (Intel + Apple Silicon)。

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

サンドボックスでインストール。Releases から .flatpak バンドルをダウンロードし、ワンコマンドでインストール。Flathub のセットアップは不要。

flatpak install --user Arroxy-*.flatpak
