Arroxy

Kostenloser 4K YouTube (+ 2000 Seiten) Downloader.
Keine Werbung, kein Bloat, kein Upselling.

Arroxy ist ein kostenloser, MIT-lizenzierter Desktop-Downloader für YouTube und 2000+ unterstützte Seiten auf Windows, macOS und Linux. Videos in bis zu 4K HDR mit 60 fps. Keine Werbung, kein Bloat, kein Upselling — einfach eine URL einfügen und loslegen.

Jede Zeile auf GitHub prüfen.
Hilfe beim ersten Start
Windows SmartScreen kann beim ersten Start "Windows protected your PC" oder "Unknown publisher" anzeigen — Arroxy ist kostenlos und Open Source, und die Windows-Builds sind nicht mit einem kostenpflichtigen Zertifikat signiert. Das gilt sowohl für Arroxy-Setup-*.exe als auch für Arroxy-Portable-*.exe und bedeutet nicht, dass Arroxy unsicher ist. Klicke auf More info und dann auf Run anyway. Lade Arroxy nur von der offiziellen GitHub Releases-Seite herunter — der Quellcode ist öffentlich, also kannst du ihn selbst prüfen oder kompilieren.
Hilfe beim ersten Start
macOS zeigt beim ersten Start eine Warnung zu einem nicht identifizierten Entwickler — Arroxy ist noch nicht signiert. Rechtsklick auf das App-Symbol → Öffnen, dann im Dialog auf Öffnen klicken. Nur einmalig.
Hilfe beim ersten Start
AppImage: Rechtsklick auf die Datei → Eigenschaften → Als Programm ausführen erlauben, oder im Terminal chmod +x Arroxy-*.AppImage ausführen. Wenn der Start scheitert, installiere libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) oder fuse2 (Arch).

Flatpak: flatpak install --user Arroxy-*.flatpak, dann aus dem App-Menü starten oder flatpak run io.github.antonio_orionus.Arroxy ausführen.

Was es kann

Alles, was du erwartest — ganz ohne Reibung.

URL einfügen, Qualität wählen, Download klicken. Mehr nicht.

Bis zu 4K UHD

2160p, 1440p, 1080p, 720p — jede Auflösung, die YouTube und andere unterstützte Seiten anbieten, plus reine Audio-Konvertierung in MP3, M4A/AAC, Opus und WAV.

60 fps & HDR erhalten

High-Framerate- und HDR-Streams kommen genau so durch, wie YouTube sie kodiert — ohne Qualitätsverlust.

Auch Playlists

Füge eine Playlist-URL ein, lade die ganze Liste herunter oder markiere nur die Videos, die du willst, bevor Arroxy sie in die Warteschlange legt.

Auto-Updates

Arroxy hält yt-dlp aktuell und liefert ffmpeg in der App mit — liefert wöchentlich Fixes, wenn sich YouTube und andere Seiten ändern.

21 Sprachen

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — erkennt deine automatisch.

Plattformübergreifend

Native Builds für Windows, macOS und Linux — Installer, Portable, DMG oder AppImage.

Untertitel, wie du willst

Manuelle oder automatisch generierte Untertitel in SRT, VTT oder ASS — neben dem Video gespeichert, in eine portable .mkv eingebettet oder in einen Subtitles/-Ordner sortiert.

SponsorBlock integriert

Überspringe oder markiere Sponsor-Segmente, Intros, Outros, Eigenwerbung und mehr — schneide sie mit FFmpeg heraus oder füge einfach Kapitel hinzu. Deine Wahl, pro Kategorie.

Automatisches Ausfüllen aus Zwischenablage

Kopiere einen beliebigen unterstützten Link irgendwo und Arroxy erkennt ihn im Moment, in dem du zurückwechselst — ein Bestätigungsdialog hält dich in Kontrolle. In den Erweiterten Einstellungen ein- oder ausschalten.

URLs automatisch bereinigen

Tracking-Parameter (si, pp, feature, utm_*, fbclid, gclid und mehr) werden automatisch aus eingefügten Links entfernt, und youtube.com/redirect-Wrapper werden aufgelöst — das URL-Feld zeigt immer den kanonischen Link.

In den Tray minimieren

Das Schließen des Fensters versteckt Arroxy im System-Tray. Downloads laufen im Hintergrund weiter — klicke auf das Tray-Symbol, um das Fenster zurückzubringen, oder beende die App über das Tray-Menü.

Eingebettete Metadaten & Cover

Titel, Upload-Datum, Künstler, Beschreibung, Cover-Art und Kapitelmarkierungen direkt in die Datei geschrieben — keine Sidecar-Dateien, kein manuelles Taggen.

YouTube + 2000 Seiten

Neben YouTube lädt Arroxy von 2000+ Seiten, die yt-dlp unterstützt — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org und viele mehr. Nur Audio und Untertitel funktionieren überall, nicht nur auf YouTube.

Privatsphäre

Was Arroxy nicht tut.

100% lokale Verarbeitung. Keine Werbung, kein Upselling, keine Drittanbieter-Server — Dateien gehen direkt von yt-dlp auf deine Festplatte.

Kein Login erforderlich

Der Standardmodus funktioniert ohne Google-Konto oder Login. In den Erweiterten Einstellungen steht optionale Cookie-Unterstützung für alters- oder mitgliedschaftsbeschränkte Inhalte zur Verfügung — standardmäßig deaktiviert.

Nur lokale Dateien

Dateien gehen direkt von yt-dlp in den Ordner, den du auswählst. Nichts wird über einen Remote-Server geleitet.

Anonyme Telemetrie

Anonyme Telemetrie über OpenPanel — eine zufällige ID pro Installation hilft, Starts, Versionen, Betriebssysteme und Abstürze zu zählen; keine URLs, Titel, Dateipfade, Kontodaten, Fingerprinting oder personenbezogenen Daten. Deine Downloads, dein Verlauf und deine Dateien verlassen nie dein Gerät.

Keine Werbung, kein Upselling

MIT-lizenziert. Keine Premium-Stufe, keine Funktionsbarrieren, keine Bannerwerbung, keine Dark Patterns. Die gesamte Pipeline läuft lokal über yt-dlp + ffmpeg.

Winget Windows

Empfohlen für Windows 10/11. Aktualisiert sich mit dem System.

winget install AntonioOrionus.Arroxy

Scoop Windows

Portable Installation via Scoop-Bucket. Keine Admin-Rechte nötig.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask hinzufügen, mit einem Befehl installieren. Universal Binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Sandboxed Installation. Lade das .flatpak-Bundle aus den Releases und installier es mit einem Befehl. Kein Flathub-Setup nötig.

flatpak install --user Arroxy-*.flatpak
