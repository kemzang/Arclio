Arroxy

Δωρεάν YouTube (+ 2000 ιστοτόποι) downloader 4K.
Χωρίς διαφημίσεις, χωρίς bloat, χωρίς upsells.

Το Arroxy είναι ένας δωρεάν desktop downloader για YouTube και 2000+ υποστηριζόμενους ιστοτόπους με άδεια MIT για Windows, macOS και Linux. Κατεβάζει βίντεο σε έως 4K HDR στα 60 fps. Χωρίς διαφημίσεις, χωρίς bloat, χωρίς upsells — απλά επικολλήστε ένα URL και ξεκινήστε.

Χωρίς διαφημίσεις Χωρίς παρακολούθηση Ανοιχτός κώδικας (MIT)

Το Arroxy είναι εφαρμογή επιτραπέζιου υπολογιστή για Windows, macOS και Linux.

Επισκεφθείτε αυτή τη σελίδα από τον υπολογιστή σας για λήψη.

Ελέγξτε κάθε γραμμή κώδικα στο GitHub.
Βοήθεια για πρώτη εκκίνηση
Το Windows SmartScreen μπορεί να εμφανίσει "Windows protected your PC" ή "Unknown publisher" κατά την πρώτη εκκίνηση — το Arroxy είναι δωρεάν και ανοιχτού κώδικα και τα Windows builds δεν είναι υπογεγραμμένα με επί πληρωμή πιστοποιητικό. Αυτό ισχύει τόσο για το Arroxy-Setup-*.exe όσο και για το Arroxy-Portable-*.exe και δεν σημαίνει ότι το Arroxy είναι επικίνδυνο. Κάντε κλικ στο More info, μετά στο Run anyway. Κατεβάστε το Arroxy μόνο από την επίσημη σελίδα GitHub Releases — ο πηγαίος κώδικας είναι δημόσιος, οπότε μπορείτε να τον ελέγξετε ή να το κατασκευάσετε μόνοι σας.
Βοήθεια για πρώτη εκκίνηση
Το macOS εμφανίζει προειδοποίηση για μη αναγνωρισμένο προγραμματιστή στην πρώτη εκκίνηση — το Arroxy δεν είναι ακόμα υπογεγραμμένο. Δεξί κλικ στο εικονίδιο → Άνοιγμα, κι έπειτα κλικ στο Άνοιγμα στο παράθυρο διαλόγου. Χρειάζεται μόνο μία φορά.
Βοήθεια για πρώτη εκκίνηση
AppImage: δεξί κλικ στο αρχείο → Ιδιότητες → Εκτέλεση ως πρόγραμμα, ή εκτελέστε chmod +x Arroxy-*.AppImage σε τερματικό. Αν η εκκίνηση αποτύχει, εγκαταστήστε libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) ή fuse2 (Arch).

Flatpak: flatpak install --user Arroxy-*.flatpak, κι έπειτα εκκινήστε από το μενού εφαρμογών ή τρέξτε flatpak run io.github.antonio_orionus.Arroxy.

Έως 4K UHD

2160p, 1440p, 1080p, 720p — κάθε ανάλυση που προσφέρουν το YouTube και οι άλλοι υποστηριζόμενοι ιστοτόποι, μαζί με μετατροπή μόνο ήχου σε MP3, M4A/AAC, Opus και WAV.

60 fps & HDR χωρίς απώλεια

Τα ρεύματα υψηλού ρυθμού καρέ και HDR μεταφέρονται ακριβώς όπως τα κωδικοποιεί το YouTube — χωρίς καμία απώλεια ποιότητας.

Και playlists

Επικόλλησε URL playlist, κατέβασε ολόκληρη τη λίστα ή τσέκαρε μόνο τα βίντεο που θέλεις πριν ο Arroxy τα βάλει στην ουρά.

Αυτόματες ενημερώσεις

Το Arroxy διατηρεί το yt-dlp ενημερωμένο και περιλαμβάνει το ffmpeg μέσα στην εφαρμογή — παρέχει διορθώσεις εβδομαδιαία καθώς το YouTube και άλλοι ιστοτόποι εξελίσσονται.

Υπότιτλοι, με τον τρόπο σας

Χειροκίνητοι ή αυτόματα δημιουργημένοι υπότιτλοι σε SRT, VTT ή ASS — αποθηκευμένοι δίπλα στο βίντεο, ενσωματωμένοι σε φορητό .mkv ή σε φάκελο Subtitles/.

SponsorBlock ενσωματωμένο

Παρακάμψτε ή σημειώστε τμήματα χορηγών, εισαγωγές, επιλόγους, αυτοπροβολές και άλλα — κόψτε τα με FFmpeg ή απλώς προσθέστε κεφάλαια. Η επιλογή σας, ανά κατηγορία.

Αυτόματη συμπλήρωση από πρόχειρο

Αντιγράψτε οποιοδήποτε υποστηριζόμενο σύνδεσμο οπουδήποτε και το Arroxy τον εντοπίζει τη στιγμή που επιστρέφετε — ένα αίτημα επιβεβαίωσης σάς κρατά σε έλεγχο. Ενεργοποιήστε ή απενεργοποιήστε στις Προχωρημένες ρυθμίσεις.

Αυτόματος καθαρισμός URL

Παράμετροι παρακολούθησης (si, pp, feature, utm_*, fbclid, gclid και άλλοι) αφαιρούνται αυτόματα από επικολλημένους συνδέσμους, και τα περιτυλίγματα youtube.com/redirect ξετυλίγονται — το πεδίο URL εμφανίζει πάντα τον κανονικό σύνδεσμο.

Ελαχιστοποίηση στο tray

Το κλείσιμο του παραθύρου κρύβει το Arroxy στο tray συστήματος. Οι λήψεις συνεχίζονται στο παρασκήνιο — κλικ στο εικονίδιο tray για να επαναφέρετε το παράθυρο, ή τερματίστε από το μενού tray.

Ενσωματωμένα μεταδεδομένα & εξώφυλλο

Τίτλος, ημερομηνία μεταφόρτωσης, καλλιτέχνης, περιγραφή, εξώφυλλο και δείκτες κεφαλαίων γράφονται απευθείας στο αρχείο — χωρίς παράπλευρα αρχεία, χωρίς χειροκίνητη ετικετοποίηση.

YouTube + 2000 ιστότοποι

Πέρα από το YouTube, το Arroxy κατεβάζει από 2000+ ιστότοπους που υποστηρίζει το yt-dlp — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org και πολλά άλλα. Μόνο ήχος και υπότιτλοι δουλεύουν παντού, όχι μόνο στο YouTube.

Απόρρητο

Τι δεν κάνει το Arroxy.

100% τοπική επεξεργασία. Χωρίς διαφημίσεις, χωρίς upsells, χωρίς διακομιστές τρίτων — τα αρχεία πηγαίνουν απευθείας από το yt-dlp στον δίσκο σας.

Δεν απαιτείται σύνδεση

Η προεπιλεγμένη λειτουργία δουλεύει χωρίς λογαριασμό Google ή σύνδεση. Προαιρετική υποστήριξη cookies είναι διαθέσιμη στις Σύνθετες ρυθμίσεις για περιεχόμενο με περιορισμένη πρόσβαση — απενεργοποιημένη από προεπιλογή.

Μόνο τοπικά αρχεία

Τα αρχεία πηγαίνουν απευθείας από το yt-dlp στον φάκελο που επιλέγετε. Τίποτα δεν δρομολογείται μέσω απομακρυσμένου διακομιστή.

Ανώνυμη τηλεμετρία

Ανώνυμη τηλεμετρία μέσω OpenPanel — ένα τυχαίο αναγνωριστικό ανά εγκατάσταση βοηθά να μετρώνται εκκινήσεις, εκδόσεις, OS και σφάλματα· χωρίς URLs, τίτλους, διαδρομές αρχείων, στοιχεία λογαριασμού, fingerprinting ή προσωπικά δεδομένα. Οι λήψεις, το ιστορικό και τα αρχεία σας δεν φεύγουν ποτέ από τη συσκευή σας.

Χωρίς διαφημίσεις, χωρίς upsells

Άδεια MIT. Χωρίς premium επίπεδο, χωρίς κλείδωμα λειτουργιών, χωρίς banner διαφημίσεις, χωρίς σκοτεινά μοτίβα. Ολόκληρη η διαδικασία εκτελείται τοπικά μέσω yt-dlp + ffmpeg.

FAQ

Συχνές ερωτήσεις

Εγκατάσταση

Επιλέξτε το κανάλι σας.

Απευθείας λήψη ή οποιοσδήποτε δημοφιλής διαχειριστής πακέτων — όλα ενημερώνονται αυτόματα σε κάθε έκδοση.

Winget Windows

Συνιστάται για Windows 10/11. Ενημερώνεται μαζί με το σύστημα.

winget install AntonioOrionus.Arroxy

Scoop Windows

Φορητή εγκατάσταση μέσω Scoop bucket. Δεν απαιτούνται δικαιώματα διαχειριστή.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Προσθέστε το tap, εγκαταστήστε με μία εντολή. Universal binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Εγκατάσταση σε sandbox. Κατεβάστε το πακέτο .flatpak από τα Releases και εγκαταστήστε με μία εντολή. Δεν χρειάζεται ρύθμιση Flathub.

flatpak install --user Arroxy-*.flatpak
