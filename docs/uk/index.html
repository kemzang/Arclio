Arroxy

Безкоштовний завантажувач YouTube (+ 2000 сайтів) 4K.
Без реклами, без зайвого, без апселів.

Arroxy — безкоштовний настільний завантажувач YouTube і 2000+ підтримуваних сайтів з ліцензією MIT для Windows, macOS і Linux. Завантажує відео до 4K HDR при 60 fps. Без реклами, без зайвого, без апселів — просто вставте посилання і готово.

Без реклами Без стеження Open source (MIT)

Arroxy — це настільний застосунок для Windows, macOS та Linux.

Відкрийте цю сторінку на комп'ютері, щоб завантажити.

Перевіряйте кожен рядок на GitHub.
Допомога при першому запуску
Windows SmartScreen може показати «Windows protected your PC» або «Unknown publisher» при першому запуску — Arroxy є безкоштовним і відкритим, а Windows-збірки не підписані платним сертифікатом. Це стосується як Arroxy-Setup-*.exe, так і Arroxy-Portable-*.exe, і не означає, що Arroxy небезпечний. Натисніть More info, потім Run anyway. Завантажуйте Arroxy лише з офіційної сторінки GitHub Releases — вихідний код відкритий, тому ви можете перевірити або зібрати його самостійно.
Допомога при першому запуску
macOS показує попередження про невизначеного розробника при першому запуску — Arroxy ще не підписаний. Правий клік по значку застосунку → Відкрити, потім натисніть Відкрити у діалозі. Лише один раз.
Допомога при першому запуску
AppImage: правий клік по файлу → Властивості → Дозволити виконання як програму, або виконайте chmod +x Arroxy-*.AppImage в терміналі. Якщо запуск усе ще не вдається, встановіть libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) або fuse2 (Arch).

Flatpak: flatpak install --user Arroxy-*.flatpak, потім запустіть з меню застосунків або виконайте flatpak run io.github.antonio_orionus.Arroxy.

Що він робить

Усе, чого ви очікуєте, без жодного тертя.

Вставте URL, оберіть якість, натисніть «Завантажити». Та й усе.

До 4K UHD

2160p, 1440p, 1080p, 720p — будь-яка роздільна здатність, яку пропонують YouTube та інші підтримувані сайти, плюс конвертація лише аудіо в MP3, M4A/AAC, Opus і WAV.

60 fps і HDR збережено

Високочастотні та HDR-потоки приходять саме такими, як їх кодує YouTube — без втрати якості.

І плейлисти теж

Встав URL плейлиста, завантаж увесь список або познач лише ті відео, які хочеш, перш ніж Arroxy поставить їх у чергу.

Автооновлення

Arroxy підтримує yt-dlp актуальним, а ffmpeg постачається всередині застосунку — щотижня випускає виправлення в міру змін YouTube та інших сайтів.

Крос-платформний

Нативні збірки для Windows, macOS та Linux — інсталятор, портативна версія, DMG або AppImage.

Субтитри — як вам зручно

Ручні або авто-згенеровані субтитри у SRT, VTT чи ASS — поруч із відео, вбудовані в портативний .mkv або складені в підпапку Subtitles/.

SponsorBlock вбудовано

Пропускайте або позначайте сегменти спонсорів, вступи, кінцівки, самопіар тощо — вирізайте через FFmpeg або додавайте розділи. Ваш вибір, для кожної категорії.

Автозаповнення з буфера обміну

Скопіюйте будь-яке підтримуване посилання будь-де, і Arroxy виявить його в момент повернення до застосунку — запит підтвердження зберігає контроль. Вмикайте та вимикайте в розширених налаштуваннях.

Автоочищення URL

Трекінгові параметри (si, pp, feature, utm_*, fbclid, gclid тощо) автоматично видаляються зі вставлених посилань, а обгортки youtube.com/redirect розгортаються — у полі URL завжди відображається канонічна адреса.

Згортання в трей

Закриття вікна ховає Arroxy в системний трей. Завантаження тривають у фоні — натисніть значок трею, щоб повернути вікно, або виходьте через меню трею.

Вбудовані метадані й обкладинка

Назва, дата завантаження, виконавець, опис, обкладинка та маркери розділів записуються безпосередньо у файл — без додаткових файлів і ручного тегування.

YouTube + 2000 сайтів

Окрім YouTube, Arroxy завантажує з 2000+ сайтів, які підтримує yt-dlp — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org та багатьох інших. Режим лише аудіо та субтитри працюють всюди, а не лише на YouTube.

Приватність

Що Arroxy не робить.

100% локальна обробка. Без реклами, без апселів, без сторонніх серверів — файли потрапляють прямо з yt-dlp на ваш диск.

Вхід не потрібен

Режим за замовчуванням працює без облікового запису або входу в Google. Опціональна підтримка кук доступна в розширених налаштуваннях для контенту з обмеженим доступом — за замовчуванням вимкнена.

Тільки локальні файли

Файли потрапляють прямо з yt-dlp у вибрану вами теку. Нічого не проходить через віддалений сервер.

Анонімна телеметрія

Анонімна телеметрія через OpenPanel — випадковий ID установки допомагає рахувати запуски, версії, OS і збої; без URL, назв, шляхів до файлів, даних акаунта, fingerprinting чи персональних даних. Ваші завантаження, історія та файли ніколи не залишають пристрій.

Без реклами, без апселів

Ліцензія MIT. Без платних рівнів, без блокування функцій, без банерної реклами, без темних паттернів. Весь конвеєр працює локально через yt-dlp + ffmpeg.

FAQ

Часті запитання

Встановлення

Оберіть канал.

Пряме завантаження або будь-який популярний менеджер пакетів — усі оновлюються автоматично з кожним релізом.

Winget Windows

Рекомендовано для Windows 10/11. Оновлюється разом із системою.

winget install AntonioOrionus.Arroxy

Scoop Windows

Портативна установка через Scoop bucket. Прав адміністратора не потрібно.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Додайте tap, встановіть однією командою. Universal Binary (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Установка в пісочниці. Скачайте бандл .flatpak з Releases і встановіть однією командою. Налаштування Flathub не потрібне.

flatpak install --user Arroxy-*.flatpak

NSIS-інсталятор, портативний .exe, .dmg, .AppImage або .flatpak — прямо з GitHub Releases.
