مساعدة الإطلاق الأول
قد يعرض Windows SmartScreen رسالة "Windows protected your PC" أو "Unknown publisher" عند الإطلاق الأول — Arroxy مجاني ومفتوح المصدر وإصدارات Windows غير موقَّعة بشهادة مدفوعة. ينطبق هذا على كلٍّ من Arroxy-Setup-*.exe وArroxy-Portable-*.exe ولا يعني ذلك بالضرورة أن Arroxy غير آمن. انقر More info، ثم انقر Run anyway. حمِّل Arroxy من صفحة GitHub Releases الرسمية فقط — المصدر عام ويمكنك مراجعته أو بناؤه بنفسك.
مساعدة الإطلاق الأول
يعرض macOS تحذير مطوّر غير معروف عند الإطلاق الأول — Arroxy لم يُوقَّع بعد. انقر بزر الفأرة الأيمن على أيقونة التطبيق ← فتح، ثم انقر فتح في مربع الحوار. مطلوب مرة واحدة فقط.
مساعدة الإطلاق الأول
AppImage: انقر بالزر الأيمن على الملف ← الخصائص ← السماح بالتنفيذ كبرنامج، أو شغّل chmod +x Arroxy-*.AppImage في الطرفية. إن لم يعمل، ثبّت libfuse2 (Ubuntu/Debian) أو fuse-libs (Fedora) أو fuse2 (Arch).

Flatpak: flatpak install --user Arroxy-*.flatpak، ثم أطلقه من قائمة التطبيقات أو شغّل flatpak run io.github.antonio_orionus.Arroxy.

حتى 4K UHD

2160p و1440p و1080p و720p — كل الدقات التي يوفّرها YouTube، مع تحويل الصوت فقط إلى MP3 وM4A/AAC وOpus وWAV.

60 إطار/ث وHDR محفوظان

تصلك بثوث معدل الإطار العالي وHDR كما شفّرها يوتيوب تماماً — بلا أي فقد في الجودة.

وقوائم التشغيل أيضاً

ألصق رابط قائمة تشغيل، ونزّل القائمة كاملة أو حدّد فقط الفيديوهات التي تريدها قبل أن يضعها Arroxy في قائمة الانتظار.

تحديثات تلقائية

يحافظ Arroxy على تحديث yt-dlp ويشحن ffmpeg داخل التطبيق — يعمل مع كل تغيير يُجريه يوتيوب.

متعدد المنصات

إصدارات أصلية لـ Windows وmacOS وLinux — مثبّت، محمول، DMG، أو AppImage.

الترجمات بطريقتك

ترجمات يدوية أو مولّدة تلقائياً بصيغ SRT وVTT وASS — احفظها بجانب الفيديو، أو ضمّنها في ملف .mkv محمول، أو ضعها في مجلد Subtitles/.

SponsorBlock مدمج

تخطَّ أو ضع علامة على مقاطع الرعاة والمقدمات والخواتم والإعلانات الذاتية وغيرها — اقطعها بـFFmpeg أو أضف فصولاً فقط. قرارك، لكل فئة.

ملء تلقائي من الحافظة

انسخ رابط يوتيوب في أي مكان ويكتشفه Arroxy فور عودتك — تأكيد موجز يبقيك في السيطرة. فعّله أو عطّله من الإعدادات المتقدمة.

تنظيف تلقائي للروابط

تُحذف معاملات التتبع (si وpp وfeature وutm_* وfbclid وgclid وغيرها) تلقائياً من روابط يوتيوب التي تلصقها، وتُكشف غلافات youtube.com/redirect — يعرض حقل الرابط دائماً الرابط الأصلي.

يختبئ في صينية النظام

يُرسل إغلاق النافذة Arroxy إلى صينية النظام. تواصل التحميلات في الخلفية — انقر أيقونة الصينية لاستعادة النافذة، أو أنهِ البرنامج من قائمة الصينية.

بيانات وصفية وغلاف مدمجان

العنوان وتاريخ الرفع والفنان والوصف وصورة الغلاف وعلامات الفصول تُكتب مباشرة في الملف — بلا ملفات مرافقة، بلا وسوم يدوية.

YouTube + 2000 موقع

إلى جانب YouTube، يقوم Arroxy بالتنزيل من أكثر من 2000 موقع يدعمها yt-dlp — Vimeo، Twitch، Twitter/X، TikTok، SoundCloud، Bandcamp، Bilibili، BBC iPlayer، archive.org، وغيرها الكثير. الصوت فقط والترجمات تعمل في كل مكان، وليس فقط على YouTube.

الخصوصية

ما لا يفعله Arroxy أبداً.

معظم محمّلات يوتيوب تطلب ملفات تعريف الارتباط في نهاية المطاف. Arroxy لن يفعل ذلك أبداً.

بلا تسجيل دخول

بلا حساب Google. بلا جلسات تنتهي. صفر خطر إبلاغ حسابك.

بلا كوكيز

يطلب Arroxy نفس الرموز التي يطلبها أي متصفح. لا يُصدَّر شيء، لا يُخزَّن شيء.

قياسات مجهولة الهوية

قياسات مجهولة الهوية عبر OpenPanel — يساعد معرّف عشوائي لكل تثبيت على عدّ مرات التشغيل والإصدارات وأنظمة التشغيل والأعطال؛ بلا URLs أو عناوين أو مسارات ملفات أو معلومات حساب أو بصمة رقمية أو بيانات شخصية. تحميلاتك وسجلك وملفاتك لا تغادر جهازك أبداً.

بلا خوادم خارجية

يعمل كامل المسار محلياً عبر yt-dlp وffmpeg. لا تلمس الملفات خادماً بعيداً قط.

التثبيت

اختر قناتك.

تحميل مباشر أو أي مدير حزم رئيسي — جميعها تُحدَّث تلقائياً مع كل إصدار.

Winget Windows

موصى به لـ Windows 10/11. يتحدث تلقائياً مع النظام.

winget install AntonioOrionus.Arroxy

Scoop Windows

تثبيت محمول عبر Scoop bucket. لا يحتاج صلاحيات مدير.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

أضف الـcask، ثبّت بأمر واحد. ثنائي عالمي (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

تثبيت في بيئة معزولة. حمّل حزمة .flatpak من Releases وثبّتها بأمر واحد. لا تحتاج إعداد Flathub.

flatpak install --user Arroxy-*.flatpak

مثبّت NSIS، ‏.exe محمول، ‏.dmg، ‏.AppImage، أو .flatpak — مباشرةً من GitHub Releases.
