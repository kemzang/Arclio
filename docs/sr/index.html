Arroxy

Бесплатни 4K преузимач за YouTube (+ 2000 сajтова).
Без реклама, без непотребног, без додатне продаје.

Arroxy је бесплатни десктоп преузимач за YouTube и 2000+ подржаних сaјтова под MIT лиценцом за Windows, macOS и Linux. Преузима видео снимке у до 4K HDR при 60 fps. Без реклама, без непотребног, без додатне продаје — само налепите URL и кренути.

Без реклама Без праћења Отворени код (MIT)

Arroxy је десктоп апликација за Windows, macOS и Linux.

Посетите ову страницу на рачунару да бисте преузели.

Прегледајте сваку линију на GitHub-у.
Помоћ при првом покретању
Windows SmartScreen може приказати «Windows protected your PC» или «Unknown publisher» при prvom покретању — Arroxy је бесплатан и отвореног кода, а Windows верзије нису потписане плаћеним сертификатом. Ово се односи на Arroxy-Setup-*.exe и Arroxy-Portable-*.exe и не значи да је Arroxy несигуран. Кликните More info, затим Run anyway. Преузимајте Arroxy искључиво са званичне GitHub Releases странице — изворни код је јаван, тако да га можете сами прегледати или компајлирати.
Помоћ при првом покретању
macOS при првом покретању приказује упозорење о непознатом програмеру — Arroxy још увек није потписан. Десни клик на икону апликације → Отвори, затим кликните Отвори у дијалогу. Потребно само једном.
Помоћ при првом покретању
AppImage: десни клик на датотеку → Својства → Дозволи извршавање као програм, или покрените chmod +x Arroxy-*.AppImage у терминалу. Ако покретање и даље не успе, инсталирајте libfuse2 (Ubuntu/Debian), fuse-libs (Fedora) или fuse2 (Arch).

Flatpak: flatpak install --user Arroxy-*.flatpak, затим покрените из менија апликација или извршите flatpak run io.github.antonio_orionus.Arroxy.

До 4K UHD

2160p, 1440p, 1080p, 720p — svaka rezolucija koju YouTube i drugi podržani sajтovi nude, plus konverzija samo zvuka u MP3, M4A/AAC, Opus i WAV.

60 fps и HDR без губитака

Потоци са великом брзином кадрова и HDR долазе тачно онако како их YouTube кодује — без губитка квалитета.

I plejliste

Nalepi URL plejliste, preuzmi celu listu ili označi samo video snimke koje želiš pre nego što ih Arroxy stavi u red.

Аутоматска ажурирања

Arroxy одржава yt-dlp ажурним, а ffmpeg стиже унутар апликације — испоручује исправке недељно у складу са изменама YouTube-а и других сajтова.

Прескочите или означите сегменте спонзора, уводе, одјаве, само-промоције и још много тога — исеците их преко FFmpeg-а или само додајте поглавља. Ваш избор, по категорији.

Аутоматско попуњавање из клипборда

Копирајте bilo koji podržani линк где год и Arroxy ће га открити чим се вратите — упит за потврду вас держи у контроли. Укључите или искључите у напредним подешавањима.

Аутоматско чишћење URL-ова

Параметри за праћење (si, pp, feature, utm_*, fbclid, gclid и остали) аутоматски се уклањају из налепљених линкова, а youtube.com/redirect омотачи се развијају — поље за URL увек приказује канонски линк.

Скривање у системску траку

Затварање прозора склања Arroxy у системску траку. Преузимања настављају да раде у позадини — кликните на икону у траци да вратите прозор, или изађите из менија траке.

Уграђени метаподаци и омот

Наслов, датум отпремања, аутор, опис, омот и маркери поглавља уписани директно у датотеку — без пратећих датотека, без ручног означавања.

YouTube + 2000 сajтова

Поред YouTube-а, Arroxy преузима са 2000+ сaјтова које yt-dlp подржава — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org и многих других. Режим само аудио и титлови раде свуда, не само на YouTube-у.

Приватност

Шта Arroxy не ради.

100% локална обрада. Без реклама, без додатне продаје, без сервера трећих страна — фajлови иду директно са yt-dlp на ваш диск.

Пријава није потребна

Подразумевани режим ради без Google налога или пријаве. Опционална подршка за колачиће доступна је у напредним подешавањима за садржај са ограниченим приступом — подразумевано искључена.

Само локалне датотеке

Фajлови иду директно са yt-dlp у фасциклу коју одаберете. Ништа не пролази кроз удаљени сервер.

Анонимна телеметрија

Анонимна телеметрија преко OpenPanel-а — насумични ID по инсталацији помаже да се броје покретања, верзије, OS и рушења; без URL-ова, наслова, путања датотека, података о налогу, fingerprinting-а или личних података. Ваша преузимања, историја и датотеке никада не напуштају ваш уређај.

Инсталација

Одаберите канал.

Директно преузимање или неки од популарних менаџера пакета — сви се аутоматски ажурирају са сваким издањем.

Winget Windows

Препоручено за Windows 10/11. Аутоматски се ажурира са системом.

winget install AntonioOrionus.Arroxy

Scoop Windows

Портабилна инсталација преко Scoop bucket-а. Нису потребна администраторска права.

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Додајте tap, инсталирајте једном командом. Универзална верзија (Intel + Apple Silicon).

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

Инсталација у изолованом окружењу. Преузмите .flatpak пакет из Releases-а и инсталирајте jednom командом. Није потребно подешавање Flathub-а.

flatpak install --user Arroxy-*.flatpak
