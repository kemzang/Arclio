مفت 4K YouTube ڈاؤن لوڈر۔
کوئی کوکیز نہیں۔ کوئی لاگ اِن نہیں۔ کوئی ٹوٹا ہوا سیشن نہیں۔

Arroxy ایک مفت، MIT لائسنس یافتہ ڈیسک ٹاپ YouTube ڈاؤن لوڈر ہے جو Windows، macOS اور Linux کے لیے ہے۔ یہ ویڈیوز کو 4K HDR تک 60 fps پر ڈاؤن لوڈ کرتا ہے — Google اکاؤنٹ، براؤزر کوکیز یا کسی بھی لاگ اِن کے بغیر۔

Windows SmartScreen پہلی بار لانچ پر "Windows protected your PC" یا "Unknown publisher" دکھا سکتا ہے — Arroxy مفت اور اوپن سورس ہے اور Windows بلڈز کو ادائیگی والے سرٹیفکیٹ سے سائن نہیں کیا گیا۔ یہ Arroxy-Setup-*.exe اور Arroxy-Portable-*.exe دونوں پر لاگو ہوتا ہے اور اس کا مطلب نہیں کہ Arroxy غیر محفوظ ہے۔ More info پر کلک کریں، پھر Run anyway پر کلک کریں۔ Arroxy صرف آفیشل GitHub Releases صفحے سے ڈاؤن لوڈ کریں — سورس کوڈ عوامی ہے، اس لیے آپ خود اسے جانچ یا بنا سکتے ہیں۔
پہلی بار لانچ کرنے میں مدد
macOS پہلی بار لانچ پر غیر شناخت شدہ ڈویلپر کی تنبیہ دکھاتا ہے — Arroxy ابھی کوڈ سائن شدہ نہیں ہے۔ ایپ آئیکن پر دائیں کلک کریں ← Open، پھر ڈائیلاگ میں Open پر کلک کریں۔ صرف ایک بار کرنا ہوگا۔
پہلی بار لانچ کرنے میں مدد
AppImage: فائل پر دائیں کلک کریں ← Properties → Allow executing as program، یا ٹرمینل میں chmod +x Arroxy-*.AppImage چلائیں۔ پھر بھی نہ چلے تو libfuse2 (Ubuntu/Debian)، fuse-libs (Fedora)، یا fuse2 (Arch) انسٹال کریں۔

Flatpak: flatpak install --user Arroxy-*.flatpak چلائیں، پھر اپنے ایپ مینو سے لانچ کریں یا flatpak run io.github.antonio_orionus.Arroxy چلائیں۔

2160p، 1440p، 1080p، 720p — ہر وہ ریزولوشن جو YouTube پیش کرتا ہے، نیز صرف آڈیو MP3، AAC اور Opus۔

60 fps اور HDR محفوظ

ہائی فریم ریٹ اور HDR اسٹریمز بالکل ویسے ہی آتے ہیں جیسے YouTube انکوڈ کرتا ہے — کوالٹی میں کوئی کمی نہیں۔

ایک ساتھ کئی

جتنے چاہیں ویڈیوز قطار میں لگائیں۔ ڈاؤن لوڈ پینل ہر ایک کی پیش رفت متوازی طور پر دکھاتا ہے۔

کلپ بورڈ خودکار بھرائی

کہیں بھی YouTube لنک کاپی کریں اور Arroxy واپس سوئچ کرتے ہی اسے پہچان لیتا ہے — ایک تصدیقی پرامپٹ آپ کو کنٹرول میں رکھتا ہے۔ ایڈوانسڈ سیٹنگز میں فعال یا غیر فعال کریں۔

خودکار طور پر URL کی صفائی

ٹریکنگ پیرامیٹرز (si، pp، feature، utm_*، fbclid، gclid اور دیگر) پیسٹ کیے گئے YouTube لنکس سے خود بخود ہٹا دیے جاتے ہیں، اور youtube.com/redirect ریپرز کھل جاتے ہیں — URL فیلڈ ہمیشہ کینونیکل لنک دکھاتا ہے۔

ٹرے میں چھپ جاتا ہے

ونڈو بند کرنے پر Arroxy آپ کے سسٹم ٹرے میں چلا جاتا ہے۔ ڈاؤن لوڈز پس منظر میں جاری رہتے ہیں — ٹرے آئیکن پر کلک کرکے ونڈو واپس لائیں، یا ٹرے مینو سے باہر نکلیں۔

ایمبیڈڈ میٹا ڈیٹا اور آرٹ

عنوان، اپلوڈ کی تاریخ، فنکار، تفصیل، کور آرٹ اور چیپٹر مارکرز براہ راست فائل میں لکھے جاتے ہیں — کوئی سائڈ کار فائلیں نہیں، کوئی دستی ٹیگنگ نہیں۔

اکثر پوچھے جانے والے سوالات

میں کن ویڈیو کوالٹیز میں ڈاؤن لوڈ کر سکتا ہوں؟
وہ سب جو YouTube پیش کرتا ہے — 4K UHD (2160p)، 1440p QHD، 1080p Full HD، 720p، 480p، 360p، اور صرف آڈیو۔ ہائی فریم ریٹ اسٹریمز (60 fps، 120 fps) اور HDR مواد جیسے ہیں ویسے ہی محفوظ رہتے ہیں۔ Arroxy ہر دستیاب فارمیٹ دکھاتا ہے اور آپ کو بالکل وہی منتخب کرنے دیتا ہے جو آپ لینا چاہتے ہیں۔
کیا یہ واقعی مفت ہے؟
جی ہاں۔ MIT لائسنس یافتہ۔ کوئی پریمیم ٹیئر نہیں، کوئی فیچر گیٹنگ نہیں۔
Arroxy کن زبانوں میں دستیاب ہے؟
اکیس، باکس سے باہر: English، Español (ہسپانوی)، Deutsch (جرمن)، Français (فرانسیسی)، 日本語 (جاپانی)، 中文 (چینی)، Русский (روسی)، Українська (یوکرینی)، हिन्दी (ہندی)، Afaan Oromoo، Kiswahili، O'zbekcha (ازبک)، Tiếng Việt (ویتنامی)، አማርኛ (امہاری)، العربية (عربی)، اردو، پښتو (پشتو)، বাংলা (بنگالی)، မြန်မာဘာသာ (برمی)، Ελληνικά (یونانی)، اور Српски (صربی)۔ Arroxy پہلی بار لانچ پر آپ کے آپریٹنگ سسٹم کی زبان خود بخود پہچان لیتا ہے اور آپ ٹول بار میں زبان منتخب کرنے والے سے کسی بھی وقت تبدیل کر سکتے ہیں۔ ترجمے src/shared/i18n/locales/ میں سادہ TypeScript آبجیکٹس کے طور پر موجود ہیں — حصہ ڈالنے کے لیے GitHub پر PR کھولیں۔
کیا مجھے کچھ انسٹال کرنے کی ضرورت ہے؟
نہیں۔ yt-dlp اور ffmpeg پہلی بار لانچ پر اپنے آفیشل GitHub releases سے خود بخود ڈاؤن لوڈ ہو جاتے ہیں اور آپ کی مشین پر کیش ہو جاتے ہیں۔ اس کے بعد، کسی اضافی سیٹ اپ کی ضرورت نہیں۔
اگر YouTube کچھ بدلے تو کیا یہ کام کرتا رہے گا؟
جی ہاں — اور Arroxy میں دو پرتیں لچک کی ہیں۔ پہلی، yt-dlp سب سے زیادہ فعال طور پر برقرار رکھے جانے والے اوپن سورس ٹولز میں سے ایک ہے — یہ YouTube کی تبدیلیوں کے گھنٹوں کے اندر اپڈیٹ ہوتا ہے۔ دوسری، Arroxy کوکیز یا آپ کے Google اکاؤنٹ پر بالکل انحصار نہیں کرتا، اس لیے کوئی سیشن ختم ہونے والا نہیں اور کوئی اسناد گردش کرنے والی نہیں۔ یہ امتزاج اسے براؤزر سے ایکسپورٹ کی گئی کوکیز پر منحصر ٹولز کے مقابلے میں نمایاں طور پر زیادہ مستحکم بناتا ہے۔
کیا میں پلے لسٹس ڈاؤن لوڈ کر سکتا ہوں؟
آج کل واحد ویڈیوز سپورٹڈ ہیں۔ پلے لسٹ اور چینل سپورٹ روڈ میپ پر ہے۔
کیا اسے میرے YouTube اکاؤنٹ یا کوکیز کی ضرورت ہے؟
نہیں — اور یہ جتنا لگتا ہے اس سے کہیں زیادہ اہم ہے۔ زیادہ تر ٹولز جو YouTube اپڈیٹ کے بعد کام کرنا بند کر دیتے ہیں، آپ سے براؤزر کی YouTube کوکیز ایکسپورٹ کرنے کو کہتے ہیں۔ یہ حل ہر ~30 منٹ بعد ٹوٹ جاتا ہے کیونکہ YouTube سیشن گردش کرتا ہے، اور yt-dlp کی اپنی دستاویزات تنبیہ کرتی ہیں کہ یہ آپ کے Google اکاؤنٹ کو فلیگ کر سکتا ہے۔ Arroxy کبھی کوکیز یا اسناد استعمال نہیں کرتا۔ کوئی لاگ اِن نہیں۔ کوئی اکاؤنٹ منسلک نہیں۔ کچھ ختم ہونے والا نہیں، کچھ بین ہونے والا نہیں۔
macOS کہتا ہے "ایپ خراب ہے" یا "نہیں کھل سکتی" — میں کیا کروں؟
یہ macOS Gatekeeper بغیر سائن کی ایپ کو بلاک کر رہا ہے — حقیقی نقصان نہیں ہے۔ macOS پر پہلی بار لانچ کے لیے README میں قدم بہ قدم ہدایات موجود ہیں۔
کیا یہ قانونی ہے؟
ذاتی استعمال کے لیے ویڈیوز ڈاؤن لوڈ کرنا زیادہ تر دائرہ اختیار میں عام طور پر قبول کیا جاتا ہے۔ آپ YouTube کی Terms of Service اور اپنے مقامی قوانین کی تعمیل کے ذمہ دار ہیں۔

Winget Windows

Windows 10/11 کے لیے تجویز کردہ۔ سسٹم کے ساتھ خودکار اپڈیٹس۔

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop bucket کے ذریعے پورٹیبل انسٹال۔ ایڈمن حقوق کی ضرورت نہیں۔

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask ٹیپ کریں، ایک کمانڈ سے انسٹال کریں۔ یونیورسل بائنری (Intel + Apple Silicon)۔

brew tap antonio-orionus/arroxy && brew install --cask arroxy
