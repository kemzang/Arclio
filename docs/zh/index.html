首次启动帮助
首次启动时，Windows SmartScreen 可能显示 «Windows protected your PC» 或 «Unknown publisher» —— Arroxy 是免费开源软件，Windows 版本未使用付费证书签名。这适用于 Arroxy-Setup-*.exe 和 Arroxy-Portable-*.exe，不代表 Arroxy 不安全。点击 More info，再点击 Run anyway。请仅从官方 GitHub Releases 页面下载 Arroxy —— 源代码公开，您可以自行审查或编译。
首次启动帮助
macOS 在首次启动时会显示未识别的开发者警告 —— Arroxy 尚未进行代码签名。右键单击应用图标 → 打开，然后在对话框中点击打开。仅需一次。
首次启动帮助
AppImage：右键单击文件 → 属性 → 允许作为程序执行，或在终端运行 chmod +x Arroxy-*.AppImage。如果仍无法启动，请安装 libfuse2 (Ubuntu/Debian)、fuse-libs (Fedora) 或 fuse2 (Arch)。

Flatpak：运行 flatpak install --user Arroxy-*.flatpak，然后从应用菜单启动或运行 flatpak run io.github.antonio_orionus.Arroxy。

它能做什么

一切如你所愿，毫无阻碍。

粘贴 URL，选择画质，点击下载。就这么简单。

高达 4K UHD

2160p、1440p、1080p、720p——YouTube 和其他支持网站提供的各档分辨率都能下，还支持仅音频转换为 MP3、M4A/AAC、Opus 和 WAV。

保留 60 fps 与 HDR

高帧率与 HDR 流原样直通，正如 YouTube 编码的那样 — 零画质损失。

也支持播放列表

粘贴播放列表链接后，可以下载整个列表，也可以先勾选想要的视频，再让 Arroxy 加入队列。

自动更新

Arroxy 会保持 yt-dlp 最新，并把 ffmpeg 随应用一起提供——每周发布修复以应对 YouTube 和其他网站的变化。

21 种语言

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — 自动检测你的语言。

跨平台

Windows、macOS、Linux 原生构建 — 安装包、便携版、DMG 或 AppImage。

字幕，随你所愿

以 SRT、VTT 或 ASS 获取手动或自动生成的字幕 — 保存到视频旁边、嵌入便携的 .mkv，或整理到 Subtitles/ 子文件夹。

内置 SponsorBlock

跳过或标记赞助商、片头、片尾、自我推广等片段 — 用 FFmpeg 剪除或直接添加章节。按类别自由选择。

剪贴板自动填写

在任何地方复制任意支持的链接，切回应用的瞬间 Arroxy 立即检测——确认提示让你保持控制。在高级设置中开启或关闭。

自动清理 URL

粘贴的链接会自动剥除跟踪参数（si、pp、feature、utm_*、fbclid、gclid 等），并解包 youtube.com/redirect 跳转链接——URL 字段始终显示规范链接。

最小化到托盘

关闭窗口后 Arroxy 将缩入系统托盘，下载在后台持续运行 — 单击托盘图标可恢复窗口，或通过托盘菜单退出应用。

嵌入元数据与封面

标题、上传日期、艺术家、描述、封面和章节标记直接写入文件——无需附属文件，无需手动标注。

YouTube + 2000 个网站

除 YouTube 外，Arroxy 还支持从 yt-dlp 覆盖的 2000+ 个网站下载——Vimeo、Twitch、Twitter/X、TikTok、SoundCloud、Bandcamp、Bilibili、BBC iPlayer、archive.org 等众多平台。仅音频和字幕在所有支持的网站上均可使用，不限于 YouTube。

隐私

Arroxy 不会做什么。

100% 本地处理。无广告、无追加销售、无第三方服务器——文件直接从 yt-dlp 保存到你的硬盘。

无需登录

默认模式无需任何 Google 账号或登录。高级设置中提供可选的 Cookie 支持，用于年龄限制或仅会员内容——默认关闭。

仅本地文件

文件直接从 yt-dlp 保存到你选择的文件夹。不经过任何远程服务器。

匿名遥测

通过 OpenPanel 进行匿名遥测 — 每次安装的随机 ID 只用于统计启动次数、版本、OS 和崩溃；不包含 URL、标题、文件路径、账号信息、指纹识别或个人数据。你的下载、历史和文件永远不会离开你的设备。

无广告、无追加销售

MIT 许可证。无付费版、无功能门槛、无横幅广告、无暗黑模式。整个流程通过 yt-dlp + ffmpeg 在本地运行。

安装

选择你的渠道。

直接下载或任意主流包管理器 — 每次发布都会自动更新。

Winget Windows

推荐用于 Windows 10/11。随系统自动更新。

winget install AntonioOrionus.Arroxy

Scoop Windows

通过 Scoop bucket 便携安装。无需管理员权限。

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

添加 cask，一行命令安装。通用二进制 (Intel + Apple Silicon)。

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

沙箱安装。从 Releases 下载 .flatpak 包，一行命令安装。无需配置 Flathub。

flatpak install --user Arroxy-*.flatpak
