Arroxy

বিনামূল্যে 4K YouTube ডাউনলোডার।
কোনো কুকি নেই। কোনো লগইন নেই। কোনো ভাঙা সেশন নেই।

Arroxy হলো Windows, macOS এবং Linux-এর জন্য একটি বিনামূল্যে, MIT-লাইসেন্সপ্রাপ্ত ডেস্কটপ YouTube ডাউনলোডার। এটি 60 fps পর্যন্ত 4K HDR-এ ভিডিও ডাউনলোড করে — কখনো Google অ্যাকাউন্ট, ব্রাউজার কুকি বা কোনো লগইন না চেয়ে।

কোনো Google অ্যাকাউন্ট নেই কোনো ট্র্যাকিং নেই ওপেন সোর্স (MIT)

Arroxy Windows, macOS এবং Linux-এর জন্য একটি ডেস্কটপ অ্যাপ।

ডাউনলোড করতে আপনার কম্পিউটারে এই পৃষ্ঠাটি দেখুন।

GitHub-এ প্রতিটি লাইন অডিট করুন।
প্রথমবার চালু করার সহায়তা
প্রথম চালুতে Windows SmartScreen "Windows protected your PC" বা "Unknown publisher" দেখাতে পারে — Arroxy বিনামূল্যে এবং ওপেন-সোর্স, এবং Windows বিল্ডগুলো পেইড সার্টিফিকেট দিয়ে সাইন করা নয়। এটি Arroxy-Setup-*.exe এবং Arroxy-Portable-*.exe উভয়ের ক্ষেত্রে প্রযোজ্য এবং এর অর্থ এই নয় যে Arroxy অনিরাপদ। More info ক্লিক করুন, তারপর Run anyway। শুধুমাত্র অফিশিয়াল GitHub Releases পেজ থেকে Arroxy ডাউনলোড করুন — সোর্স কোড পাবলিক, তাই আপনি নিজে পরীক্ষা করতে বা বিল্ড করতে পারবেন।
প্রথমবার চালু করার সহায়তা
প্রথম চালুতে macOS একটি অপরিচিত ডেভেলপার সতর্কতা দেখায় — Arroxy এখনো কোড-সাইন করা নয়। অ্যাপ আইকনে ডান-ক্লিক করুন → Open, তারপর ডায়ালগে Open ক্লিক করুন। শুধুমাত্র একবার দরকার।
প্রথমবার চালু করার সহায়তা
AppImage: ফাইলে ডান-ক্লিক করুন → Properties → Allow executing as program, অথবা টার্মিনালে chmod +x Arroxy-*.AppImage চালান। তবুও চালু না হলে libfuse2 (Ubuntu/Debian), fuse-libs (Fedora), বা fuse2 (Arch) ইনস্টল করুন।

Flatpak: flatpak install --user Arroxy-*.flatpak, তারপর অ্যাপ মেনু থেকে চালু করুন বা flatpak run io.github.antonio_orionus.Arroxy চালান।

Arroxy yt-dlp এবং ffmpeg সর্বদা আপ টু ডেট রাখে — YouTube-এর প্রতিটি পরিবর্তনের সাথে কাজ করে।

ক্রস-প্ল্যাটফর্ম

Windows, macOS এবং Linux-এর জন্য নেটিভ বিল্ড — ইনস্টলার, পোর্টেবল, DMG, বা AppImage।

আপনার মতো সাবটাইটেল

SRT, VTT, বা ASS-এ ম্যানুয়াল বা অটো-জেনারেটেড ক্যাপশন — ভিডিওর পাশে সেভ করুন, পোর্টেবল .mkv-তে এমবেড করুন, বা Subtitles/ ফোল্ডারে রাখুন।

SponsorBlock অন্তর্নির্মিত

স্পনসর সেগমেন্ট, ইন্ট্রো, আউট্রো, সেলফ-প্রমো এবং আরও বেশি স্কিপ বা চিহ্নিত করুন — FFmpeg দিয়ে কেটে ফেলুন বা শুধু চ্যাপ্টার যোগ করুন। প্রতিটি ক্যাটাগরিতে আপনার সিদ্ধান্ত।

ক্লিপবোর্ড অটো-ফিল

যেকোনো জায়গায় YouTube লিঙ্ক কপি করুন এবং ফিরে আসার মুহূর্তে Arroxy সেটা শনাক্ত করে — একটি নিশ্চিতকরণ প্রম্পট আপনাকে নিয়ন্ত্রণে রাখে। Advanced settings-এ চালু বা বন্ধ করুন।

URL স্বয়ংক্রিয় পরিষ্কার

ট্র্যাকিং প্যারামিটার (si, pp, feature, utm_*, fbclid, gclid এবং আরও) পেস্ট করা YouTube লিঙ্ক থেকে স্বয়ংক্রিয়ভাবে সরানো হয়, এবং youtube.com/redirect র‍্যাপার আনর‍্যাপ করা হয় — URL ফিল্ড সবসময় ক্যানোনিকাল লিঙ্ক দেখায়।

ট্রেতে লুকিয়ে যায়

উইন্ডো বন্ধ করলে Arroxy সিস্টেম ট্রেতে চলে যায়। ডাউনলোড ব্যাকগ্রাউন্ডে চলতে থাকে — উইন্ডো ফিরে আনতে ট্রে আইকনে ক্লিক করুন, বা ট্রে মেনু থেকে বের হন।

এমবেডেড মেটাডেটা ও আর্ট

শিরোনাম, আপলোড তারিখ, শিল্পী, বিবরণ, কভার আর্ট এবং চ্যাপ্টার মার্কার সরাসরি ফাইলে লেখা হয় — কোনো সাইডকার ফাইল নেই, কোনো ম্যানুয়াল ট্যাগিং নেই।

কোনো লগইন নেই

কোনো Google অ্যাকাউন্ট নেই। মেয়াদোত্তীর্ণ সেশন নেই। আপনার অ্যাকাউন্ট ফ্ল্যাগ হওয়ার শূন্য ঝুঁকি।

কোনো কুকি নেই

Arroxy যেকোনো ব্রাউজারের মতো একই টোকেন অনুরোধ করে। কিছু এক্সপোর্ট হয় না, কিছু সংরক্ষিত হয় না।

কোনো ব্যবহারকারী ID নেই

Aptabase-এর মাধ্যমে বেনামী, সেশন-কেবল টেলিমেট্রি — কোনো ইনস্টল ID নেই, কোনো ফিঙ্গারপ্রিন্টিং নেই, কোনো ব্যক্তিগত তথ্য নেই। আপনার ডাউনলোড, ইতিহাস এবং ফাইল কখনো আপনার মেশিন ছেড়ে যায় না।

কোনো তৃতীয় পক্ষের সার্ভার নেই

পুরো পাইপলাইন yt-dlp + ffmpeg-এর মাধ্যমে স্থানীয়ভাবে চলে। ফাইল কখনো রিমোট সার্ভার স্পর্শ করে না।

FAQ

প্রায়শই জিজ্ঞাসিত প্রশ্ন

আমি কোন ভিডিও মান ডাউনলোড করতে পারি?
YouTube যা দেয় সব — 4K UHD (2160p), 1440p QHD, 1080p Full HD, 720p, 480p, 360p, এবং শুধু-অডিও। হাই ফ্রেম-রেট স্ট্রিম (60 fps, 120 fps) এবং HDR কন্টেন্ট যেমন আছে তেমনই থাকে। Arroxy প্রতিটি উপলব্ধ ফরম্যাট দেখায় এবং ঠিক কোনটা নেবেন তা বেছে নিতে দেয়।
এটি কি সত্যিই বিনামূল্যে?
হ্যাঁ। MIT লাইসেন্স। কোনো প্রিমিয়াম টিয়ার নেই, কোনো ফিচার গেটিং নেই।
Arroxy কোন কোন ভাষায় পাওয়া যায়?
একুশটি, বাক্সের বাইরে: English, Español (স্পেনিশ), Deutsch (জার্মান), Français (ফরাসি), 日本語 (জাপানি), 中文 (চীনা), Русский (রাশিয়ান), Українська (ইউক্রেনিয়ান), हिन्दी (হিন্দি), Afaan Oromoo, Kiswahili, O'zbekcha (উজবেক), Tiếng Việt (ভিয়েতনামি), አማርኛ (আমহারিক), العربية (আরবি), اردو (উর্দু), پښتو (পশতু), বাংলা, မြန်မာဘာသာ (বার্মিজ), Ελληνικά (গ্রিক), এবং Српски (সার্বিয়ান)। Arroxy প্রথম চালুতে আপনার অপারেটিং সিস্টেমের ভাষা স্বয়ংক্রিয়ভাবে শনাক্ত করে এবং আপনি টুলবারের ভাষা পিকার থেকে যেকোনো সময় পরিবর্তন করতে পারেন। অনুবাদগুলো src/shared/i18n/locales/-এ সাধারণ TypeScript অবজেক্ট হিসেবে আছে — অবদান রাখতে GitHub-এ একটি PR খুলুন।
আমাকে কিছু ইনস্টল করতে হবে?
না। yt-dlp এবং ffmpeg প্রথম চালুতে তাদের অফিশিয়াল GitHub releases থেকে স্বয়ংক্রিয়ভাবে ডাউনলোড হয় এবং আপনার মেশিনে ক্যাশ হয়। এরপর কোনো অতিরিক্ত সেটআপ দরকার নেই।
YouTube কিছু পরিবর্তন করলে এটি কি কাজ করতে থাকবে?
হ্যাঁ — এবং Arroxy-তে দুটি স্থিতিস্থাপকতার স্তর আছে। প্রথমত, yt-dlp সবচেয়ে সক্রিয়ভাবে রক্ষণাবেক্ষণ করা ওপেন-সোর্স টুলগুলোর মধ্যে একটি — YouTube পরিবর্তনের ঘণ্টার মধ্যে আপডেট হয়। দ্বিতীয়ত, Arroxy কুকি বা আপনার Google অ্যাকাউন্টের উপর মোটেও নির্ভর করে না, তাই মেয়াদোত্তীর্ণ হওয়ার কোনো সেশন নেই এবং পরিবর্তন করার কোনো ক্রেডেনশিয়াল নেই। এই সংমিশ্রণ এটাকে এক্সপোর্ট করা ব্রাউজার কুকির উপর নির্ভরশীল টুলের চেয়ে উল্লেখযোগ্যভাবে বেশি স্থিতিশীল করে।
আমি কি প্লেলিস্ট ডাউনলোড করতে পারি?
আজ একক ভিডিও সমর্থিত। প্লেলিস্ট এবং চ্যানেল সাপোর্ট রোডম্যাপে আছে।
এটি কি আমার YouTube অ্যাকাউন্ট বা কুকি দরকার?
না — এবং এটা যতটা মনে হয় তার চেয়ে বড় ব্যাপার। বেশিরভাগ টুল যেগুলো YouTube আপডেটের পরে কাজ বন্ধ করে দেয়, সেগুলো আপনাকে ব্রাউজারের YouTube কুকি এক্সপোর্ট করতে বলে। এই সমাধানটি প্রতি ~৩০ মিনিটে ভেঙে পড়ে কারণ YouTube সেশন পরিবর্তন করে, এবং yt-dlp-এর নিজের ডকস সতর্ক করে যে এটি আপনার Google অ্যাকাউন্ট ফ্ল্যাগ করাতে পারে। Arroxy কখনো কুকি বা ক্রেডেনশিয়াল ব্যবহার করে না। কোনো লগইন নেই। কোনো অ্যাকাউন্ট লিঙ্ক নেই। মেয়াদোত্তীর্ণ হওয়ার কিছু নেই, নিষিদ্ধ হওয়ার কিছু নেই।
macOS বলছে "অ্যাপটি ক্ষতিগ্রস্ত" বা "খোলা যাচ্ছে না" — আমি কী করব?
এটি macOS Gatekeeper একটি আনসাইন্ড অ্যাপ ব্লক করছে — আসল ক্ষতি নয়। README-এ macOS-এ প্রথমবার চালুর জন্য ধাপে ধাপে নির্দেশ আছে।
এটি কি আইনি?
ব্যক্তিগত ব্যবহারের জন্য ভিডিও ডাউনলোড করা বেশিরভাগ বিচারক্ষেত্রে সাধারণত গ্রহণযোগ্য। YouTube-এর Terms of Service এবং আপনার স্থানীয় আইন মেনে চলা আপনার দায়িত্ব।
ইনস্টল

আপনার চ্যানেল বেছে নিন।

সরাসরি ডাউনলোড বা যেকোনো প্রধান প্যাকেজ ম্যানেজার — সব প্রতিটি রিলিজে স্বয়ংক্রিয়ভাবে আপডেট হয়।

Winget Windows

Windows 10/11-এর জন্য প্রস্তাবিত। সিস্টেমের সাথে স্বয়ংক্রিয় আপডেট।

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop bucket-এর মাধ্যমে পোর্টেবল ইনস্টল। অ্যাডমিন অধিকার দরকার নেই।

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask ট্যাপ করুন, একটি কমান্ডে ইনস্টল করুন। ইউনিভার্সাল বাইনারি (Intel + Apple Silicon)।

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

স্যান্ডবক্সড ইনস্টল। Releases থেকে .flatpak বান্ডেল ডাউনলোড করুন, একটি কমান্ডে ইনস্টল করুন। Flathub সেটআপ দরকার নেই।

flatpak install --user Arroxy-*.flatpak
