Arroxy

বিনামূল্যে 4K YouTube (+ ২০০০ সাইট) ডাউনলোডার।
কোনো বিজ্ঞাপন নেই, কোনো ব্লোট নেই, কোনো আপসেল নেই।

Arroxy হলো Windows, macOS এবং Linux-এর জন্য একটি বিনামূল্যে, MIT-লাইসেন্সপ্রাপ্ত ডেস্কটপ ডাউনলোডার — YouTube ও ২০০০+ সমর্থিত সাইটের জন্য। 60 fps পর্যন্ত 4K HDR-এ ভিডিও ডাউনলোড করে। কোনো বিজ্ঞাপন নেই, কোনো ব্লোট নেই, কোনো আপসেল নেই — শুধু URL পেস্ট করুন এবং শুরু করুন।

কোনো বিজ্ঞাপন নেই কোনো ট্র্যাকিং নেই ওপেন সোর্স (MIT)

Arroxy Windows, macOS এবং Linux-এর জন্য একটি ডেস্কটপ অ্যাপ।

ডাউনলোড করতে আপনার কম্পিউটারে এই পৃষ্ঠাটি দেখুন।

GitHub-এ প্রতিটি লাইন অডিট করুন।
প্রথমবার চালু করার সহায়তা
প্রথম চালুতে Windows SmartScreen "Windows protected your PC" বা "Unknown publisher" দেখাতে পারে — Arroxy বিনামূল্যে এবং ওপেন-সোর্স, এবং Windows বিল্ডগুলো পেইড সার্টিফিকেট দিয়ে সাইন করা নয়। এটি Arroxy-Setup-*.exe এবং Arroxy-Portable-*.exe উভয়ের ক্ষেত্রে প্রযোজ্য এবং এর অর্থ এই নয় যে Arroxy অনিরাপদ। More info ক্লিক করুন, তারপর Run anyway। শুধুমাত্র অফিশিয়াল GitHub Releases পেজ থেকে Arroxy ডাউনলোড করুন — সোর্স কোড পাবলিক, তাই আপনি নিজে পরীক্ষা করতে বা বিল্ড করতে পারবেন।
প্রথমবার চালু করার সহায়তা
প্রথম চালুতে macOS একটি অপরিচিত ডেভেলপার সতর্কতা দেখায় — Arroxy এখনো কোড-সাইন করা নয়। অ্যাপ আইকনে ডান-ক্লিক করুন → Open, তারপর ডায়ালগে Open ক্লিক করুন। শুধুমাত্র একবার দরকার।
প্রথমবার চালু করার সহায়তা
AppImage: ফাইলে ডান-ক্লিক করুন → Properties → Allow executing as program, অথবা টার্মিনালে chmod +x Arroxy-*.AppImage চালান। তবুও চালু না হলে libfuse2 (Ubuntu/Debian), fuse-libs (Fedora), বা fuse2 (Arch) ইনস্টল করুন।

Flatpak: flatpak install --user Arroxy-*.flatpak, তারপর অ্যাপ মেনু থেকে চালু করুন বা flatpak run io.github.antonio_orionus.Arroxy চালান।

এটি কী করে

যা প্রত্যাশা করেন সব আছে, কোনো ঝামেলা নেই।

URL পেস্ট করুন, মান বেছে নিন, ডাউনলোড ক্লিক করুন। এটুকুই।

4K UHD পর্যন্ত

2160p, 1440p, 1080p, 720p — YouTube ও অন্যান্য সমর্থিত সাইট যে resolution দেয়, সেগুলো সব, এবং শুধু অডিওর জন্য MP3, M4A/AAC, Opus ও WAV conversion।

60 fps ও HDR সংরক্ষিত

হাই ফ্রেম-রেট এবং HDR স্ট্রিম ঠিক যেভাবে YouTube এনকোড করে সেভাবেই আসে — কোনো মান ক্ষতি নেই।

Playlist-ও

playlist URL পেস্ট করুন, পুরো তালিকা ডাউনলোড করুন, বা Arroxy queue করার আগে শুধু যেসব ভিডিও চান সেগুলো টিক দিন।

স্বয়ংক্রিয় আপডেট

Arroxy yt-dlp আপ টু ডেট রাখে এবং ffmpeg অ্যাপের সাথেই দেয় — YouTube-এর প্রতিটি পরিবর্তনের সাথে কাজ করে।

২১টি ভাষা

English, Español, Deutsch, Français, 日本語, 中文, Русский, Українська, हिन्दी, Afaan Oromoo, Kiswahili, O'zbekcha, Tiếng Việt, አማርኛ, العربية, اردو, پښتو, বাংলা, မြန်မာဘာသာ, Ελληνικά, Српски — স্বয়ংক্রিয়ভাবে আপনার ভাষা শনাক্ত করে।

ক্রস-প্ল্যাটফর্ম

Windows, macOS এবং Linux-এর জন্য নেটিভ বিল্ড — ইনস্টলার, পোর্টেবল, DMG, বা AppImage।

আপনার মতো সাবটাইটেল

SRT, VTT, বা ASS-এ ম্যানুয়াল বা অটো-জেনারেটেড ক্যাপশন — ভিডিওর পাশে সেভ করুন, পোর্টেবল .mkv-তে এমবেড করুন, বা Subtitles/ ফোল্ডারে রাখুন।

SponsorBlock অন্তর্নির্মিত

স্পনসর সেগমেন্ট, ইন্ট্রো, আউট্রো, সেলফ-প্রমো এবং আরও বেশি স্কিপ বা চিহ্নিত করুন — FFmpeg দিয়ে কেটে ফেলুন বা শুধু চ্যাপ্টার যোগ করুন। প্রতিটি ক্যাটাগরিতে আপনার সিদ্ধান্ত।

ক্লিপবোর্ড অটো-ফিল

যেকোনো জায়গায় YouTube লিঙ্ক কপি করুন এবং ফিরে আসার মুহূর্তে Arroxy সেটা শনাক্ত করে — একটি নিশ্চিতকরণ প্রম্পট আপনাকে নিয়ন্ত্রণে রাখে। Advanced settings-এ চালু বা বন্ধ করুন।

URL স্বয়ংক্রিয় পরিষ্কার

ট্র্যাকিং প্যারামিটার (si, pp, feature, utm_*, fbclid, gclid এবং আরও) পেস্ট করা YouTube লিঙ্ক থেকে স্বয়ংক্রিয়ভাবে সরানো হয়, এবং youtube.com/redirect র‍্যাপার আনর‍্যাপ করা হয় — URL ফিল্ড সবসময় ক্যানোনিকাল লিঙ্ক দেখায়।

ট্রেতে লুকিয়ে যায়

উইন্ডো বন্ধ করলে Arroxy সিস্টেম ট্রেতে চলে যায়। ডাউনলোড ব্যাকগ্রাউন্ডে চলতে থাকে — উইন্ডো ফিরে আনতে ট্রে আইকনে ক্লিক করুন, বা ট্রে মেনু থেকে বের হন।

এমবেডেড মেটাডেটা ও আর্ট

শিরোনাম, আপলোড তারিখ, শিল্পী, বিবরণ, কভার আর্ট এবং চ্যাপ্টার মার্কার সরাসরি ফাইলে লেখা হয় — কোনো সাইডকার ফাইল নেই, কোনো ম্যানুয়াল ট্যাগিং নেই।

YouTube + 2000 সাইট

YouTube ছাড়াও, Arroxy yt-dlp সমর্থিত 2000+ সাইট থেকে ডাউনলোড করে — Vimeo, Twitch, Twitter/X, TikTok, SoundCloud, Bandcamp, Bilibili, BBC iPlayer, archive.org এবং আরও অনেক। শুধু-অডিও এবং সাবটাইটেল সর্বত্র কাজ করে, শুধু YouTube-এ নয়।

এটি কার্যক্ষেত্রে দেখুন

স্পষ্টতার জন্য তৈরি, বিশৃঙ্খলার জন্য নয়।

পছন্দ হচ্ছে?

বিনামূল্যে ডাউনলোড করুন। কোনো অ্যাকাউন্ট নেই, কোনো বিজ্ঞাপন নেই, কোনো শর্ত নেই।

গোপনীয়তা

Arroxy যা করে না।

বেশিরভাগ YouTube ডাউনলোডার শেষ পর্যন্ত আপনার কুকি চায়। Arroxy কখনো চাইবে না।

কোনো লগইন নেই

কোনো Google অ্যাকাউন্ট নেই। মেয়াদোত্তীর্ণ সেশন নেই। আপনার অ্যাকাউন্ট ফ্ল্যাগ হওয়ার শূন্য ঝুঁকি।

কোনো কুকি নেই

Arroxy যেকোনো ব্রাউজারের মতো একই টোকেন অনুরোধ করে। কিছু এক্সপোর্ট হয় না, কিছু সংরক্ষিত হয় না।

বেনামী টেলিমেট্রি

OpenPanel-এর মাধ্যমে বেনামী টেলিমেট্রি — প্রতি-ইনস্টলের একটি এলোমেলো ID চালু হওয়া, সংস্করণ, OS ও ক্র্যাশ গোনায় সাহায্য করে; কোনো URL, শিরোনাম, ফাইল পাথ, অ্যাকাউন্ট তথ্য, ফিঙ্গারপ্রিন্টিং বা ব্যক্তিগত ডেটা নেই। আপনার ডাউনলোড, ইতিহাস ও ফাইল কখনো আপনার মেশিন ছাড়ে না।

কোনো তৃতীয় পক্ষের সার্ভার নেই

পুরো পাইপলাইন yt-dlp + ffmpeg-এর মাধ্যমে স্থানীয়ভাবে চলে। ফাইল কখনো রিমোট সার্ভার স্পর্শ করে না।

প্রায়শই জিজ্ঞাসিত প্রশ্ন

ইনস্টল

আপনার চ্যানেল বেছে নিন।

সরাসরি ডাউনলোড বা যেকোনো প্রধান প্যাকেজ ম্যানেজার — সব প্রতিটি রিলিজে স্বয়ংক্রিয়ভাবে আপডেট হয়।

Winget Windows

Windows 10/11-এর জন্য প্রস্তাবিত। সিস্টেমের সাথে স্বয়ংক্রিয় আপডেট।

winget install AntonioOrionus.Arroxy

Scoop Windows

Scoop bucket-এর মাধ্যমে পোর্টেবল ইনস্টল। অ্যাডমিন অধিকার দরকার নেই।

scoop bucket add arroxy https://github.com/antonio-orionus/scoop-bucket && scoop install arroxy

Homebrew macOS

Cask ট্যাপ করুন, একটি কমান্ডে ইনস্টল করুন। ইউনিভার্সাল বাইনারি (Intel + Apple Silicon)।

brew tap antonio-orionus/arroxy && brew install --cask arroxy

Flatpak Linux

স্যান্ডবক্সড ইনস্টল। Releases থেকে .flatpak বান্ডেল ডাউনলোড করুন, একটি কমান্ডে ইনস্টল করুন। Flathub সেটআপ দরকার নেই।

flatpak install --user Arroxy-*.flatpak
